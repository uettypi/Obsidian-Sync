## 1.2026ciscn&ccb分区赛 easy_time
尝试做了一下学长们夺冠的2026ciscn&ccb分区赛的Web题——easy_time，以下是我的Write Up。

1. 首先先审计index.py中关于login页面的代码段，可知h2应为密码md5两次之后的结果，故利用cmd5这个网站破解即可得到密码应为secret。![image.png](https://img.uettypi.top/2026/04/20260406213214799.png)
2. 成功登录进入后台后，有3个路径，粗略地看了一下，如果有漏洞的话，大概是：插件上传——文件上传、留言板——SSTI/XSS、个人About/头像——SSRF&文件上传。然后，开始审计相关功能的代码。审完后，可以发现，所有用到模板的地方都是传参，没有拼接，可见没有SSTI；然后是SSRF，这个有，但是由于这段代码限制了协议，所以各类伪协议都不能用了：
```python
	if parsed.scheme not in {"http", "https"}:
		return None
```
3. 最后是文件上传，这里有两个点，一个是插件上传，另一个是头像上传，插件上传不太熟，先看头像上传，看起来是要上传一个图片马，但后缀名是白名单，且强制重命名，所以这里线索断了，只能看插件上传了。
