# 周报（20260518-20260524）
## 1.CISCN2021 Quals filter
1. 在附件里的`SiteController.php`中找到了如下的一段代码：
```php
public function actionIndex()

{

$file = Yii::$app->request->get('file');

$res = file_get_contents($file);

file_put_contents($file,$res);

return $this->render('index');

}
```
看到`file_get_contents()`和`file_put_contents()`这两个高危函数了，接下来研究一下能否利用。
2. 参数`file`可控，变量`$res`在读取了`$file`的值后，又将`$res`的值重新写回了变量`$file`。这里乍一看分明是在脱裤子放屁，但这正好给了我们可乘之机，就是Filter Chain。这也正好呼应了题目名“filter”。
