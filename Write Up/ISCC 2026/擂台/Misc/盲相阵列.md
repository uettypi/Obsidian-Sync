
## 分析过程

阅读 `rx_note.txt` 文件，里面明确提供了系统的关键参数，包括采样率、符号频率、残余载波频偏的近似估值、训练序列的长度以及后续数据位处理的具体步骤。

```text
Blind Phase Array receiver note
===============================
The dumped CSV contains complex I/Q samples after a faulty downconverter.
Sample rate: 48000 Hz. Nominal symbol rate: 1200 baud.
The residual carrier offset is close to +1700 Hz.
The symbol boundary is not at sample 0; find it with the training block.
The first 80 recovered symbols are training/sync symbols.
Payload path from the old FPGA notes: PN mask -> 16-lane column DMA -> H(7,4) nibbles -> BP1 frame.
Selected codewords are expected to contain one bad bit. CRC32 at the frame tail is the final integrity check.
```

根据上述说明可以计算出，单符号对应的采样点数为 `48000 / 1200 = 40`，残余载波需要以 `+1700 Hz` 的频率进行下变频来实施补偿。对于 BPSK 调制中存在的未知相位偏移，可利用对复数抽样求平方均值的方法来抹除符号极性的影响：具体操作是对抽取出的复数符号执行 `sum(symbol^2)` 运算，所得角度的一半即为经载波补偿后的全局参考相位。训练序列的前 64 位由呈交替状的 `1010...` 构成，主要用于精准定位符号边界并判定极性；紧接着的 16 位则代表 PN 码的初始种子。具体探测到的数据情况见下文：

在执行解码操作时，我们需要从训练块结束的地方开始提取有效载荷（payload），提取的比特总数应保证为 `16 * 7` 的整数倍，当前附件提取出了 560 位有效数据。PN 序列采用 16 位的线性反馈移位寄存器（LFSR）生成，反馈多项式配置为 `0x1d00`。为了保证普适性，解题脚本直接从同步数据段中动态获取种子（seed），而非固定写入 `0xe5a2`；此外，通过不断尝试匹配 BP1 的帧头部特征以及 CRC32 校验和，可以自动化推断出 PN 序列的起始偏移量（当前附件算出的 skip 值为 8）。剥离 PN 掩码后，我们需要对 16 路列级 DMA 执行逆向重排，即把接收到的 16 列数据按照行的顺序重新读取：矩阵变换逻辑为 `out[row * 16 + lane] = in[lane * rows + row]`。随后，将数据按照每 7 位一组送入 H(7,4) 解码模块。这里的码字结构定义为 `[p1, p2, d1, p4, d2, d3, d4]`，其校验位的逻辑关系是：`p1=d1^d2^d4`、`p2=d1^d3^d4`、`p4=d2^d3^d4`。最终整个 payload 能够输出 80 个解码后的码字，经过验证，其中 75 个码字完全正确，另有 5 个码字触发了 1 比特的自动纠错，这与提示文档中提到的“预计选定的码字中包含一个错误位”完全吻合。

经过 Hamming 解码还原出的完整数据帧以特征字符串 `BP1 ` 作为起始标志，紧随其后的是有效正文内容，最后以 4 个字节的大端序 CRC32 校验和收尾。对起始头 `BP1 ` 之后、末尾 CRC 之前的数据段进行 CRC32 运算，所得的哈希值 `0xa60a993c` 与帧尾携带的 `a6 0a 99 3c` 完美对应，由此确认提取出的正文 `ISCC{R7!q_Z@4m^T9?p$V%k&2*n~L#x}` 即为本题最终答案。

为了提升代码的适应能力，本脚本在参数提取上进行了通用化设计：像采样率、符号率、残载频偏估值、训练序列尺寸及 DMA 通道数等参数，均通过解析 `rx_note.txt` 获取；I/Q 原始信号自 `array_iq.csv` 载入；符号起始边界利用开头 64 位交替出现的训练符号进行滑动扫描来确认；PN 初始种子由训练块结尾的 16 位推导而出；有效载荷的截取长度会自动匹配 `16 * 7` 的对齐要求；PN 序列跳过的位数（skip）利用 BP1 帧头特征与 CRC32 验证机制进行动态爆破确定；最终的 flag 则从校验通过的 BP1 帧内部截出。这样一来，若遇到相同题型但不同数据的其他附件，诸如样本本体、起始边界、随机种子、载荷长度以及校验值等要素均会根据新文件实时演算，从而彻底摒弃了对固定硬编码偏移量或明文特征的依赖。

```python
import csv
import io
import math
import os
import re
import sys
import zipfile
import zlib


def read_bundle(path):
    note_text = ""
    csv_text = ""
    if zipfile.is_zipfile(path):
        with zipfile.ZipFile(path) as zf:
            names = zf.namelist()
            note_name = next((n for n in names if n.replace('\\', '/').endswith('rx_note.txt')), None)
            csv_name = next((n for n in names if n.replace('\\', '/').endswith('array_iq.csv')), None)
            if note_name is None or csv_name is None:
                raise ValueError('missing rx_note.txt or array_iq.csv')
            note_text = zf.read(note_name).decode('utf-8', errors='replace')
            csv_text = zf.read(csv_name).decode('utf-8', errors='replace')
    elif os.path.isdir(path):
        note_path = None
        csv_path = None
        for root, _, files in os.walk(path):
            for name in files:
                full = os.path.join(root, name)
                if name == 'rx_note.txt':
                    note_path = full
                elif name == 'array_iq.csv':
                    csv_path = full
        if note_path is None or csv_path is None:
            raise ValueError('missing rx_note.txt or array_iq.csv')
        with open(note_path, 'r', encoding='utf-8', errors='replace') as f:
            note_text = f.read()
        with open(csv_path, 'r', encoding='utf-8', errors='replace') as f:
            csv_text = f.read()
    else:
        with open(path, 'r', encoding='utf-8', errors='replace') as f:
            csv_text = f.read()
    return note_text, csv_text


def parse_note(note_text):
    def grab(pattern, default=None, cast=float):
        m = re.search(pattern, note_text, re.I)
        if not m:
            if default is None:
                raise ValueError('missing parameter in rx_note.txt')
            return default
        return cast(m.group(1))

    sample_rate = grab(r'Sample\s+rate:\s*([0-9.]+)', cast=float)
    symbol_rate = grab(r'symbol\s+rate:\s*([0-9.]+)', cast=float)
    carrier = grab(r'offset\s+is\s+close\s+to\s*([+-]?[0-9.]+)', cast=float)
    training = grab(r'first\s+([0-9]+)\s+recovered\s+symbols', default=80, cast=int)
    lanes = grab(r'([0-9]+)\s*-\s*lane\s+column\s+DMA', default=16, cast=int)
    return sample_rate, symbol_rate, carrier, training, lanes


def read_iq(csv_text):
    out = []
    reader = csv.DictReader(io.StringIO(csv_text))
    for row in reader:
        out.append(complex(float(row['i']), float(row['q'])))
    if not out:
        raise ValueError('empty csv')
    return out


def lfsr_mask(seed, count):
    state = seed & 0xffff
    poly = 0x1d00
    out = []
    for _ in range(count):
        out_bit = state & 1
        feedback = (state & poly).bit_count() & 1
        state = (state >> 1) | (feedback << 15)
        out.append(out_bit)
    return out


def hamming_codewords():
    table = []
    for n in range(16):
        d1 = (n >> 3) & 1
        d2 = (n >> 2) & 1
        d3 = (n >> 1) & 1
        d4 = n & 1
        p1 = d1 ^ d2 ^ d4
        p2 = d1 ^ d3 ^ d4
        p4 = d2 ^ d3 ^ d4
        table.append([p1, p2, d1, p4, d2, d3, d4])
    return table


def hamming_decode(bits):
    words = hamming_codewords()
    nibbles = []
    errors = 0
    for i in range(0, len(bits), 7):
        word = bits[i:i + 7]
        if len(word) < 7:
            break
        best_dist = 8
        best_nibble = 0
        for nibble, codeword in enumerate(words):
            dist = sum(a != b for a, b in zip(word, codeword))
            if dist < best_dist:
                best_dist = dist
                best_nibble = nibble
        errors += best_dist
        nibbles.append(best_nibble)
    data = bytearray()
    for i in range(0, len(nibbles) - 1, 2):
        data.append((nibbles[i] << 4) | nibbles[i + 1])
    return bytes(data), errors


def undo_column_dma(bits, lanes):
    rows = len(bits) // lanes
    out = []
    for r in range(rows):
        for lane in range(lanes):
            out.append(bits[lane * rows + r])
    return out


def try_decode_symbol_bits(bits, training, lanes):
    if len(bits) <= training + lanes * 7:
        return None
    sync_bits = bits[64:training]
    if len(sync_bits) != 16:
        return None
    seed = int(''.join(str(b) for b in sync_bits), 2)
    payload = bits[training:]
    block = lanes * 7
    usable = (len(payload) // block) * block
    payload = payload[:usable]
    if usable == 0:
        return None

    max_skip = max(128, training + 64)
    mask = lfsr_mask(seed, usable + max_skip)
    for skip in range(max_skip):
        unmasked = [payload[i] ^ mask[i + skip] for i in range(usable)]
        ordered = undo_column_dma(unmasked, lanes)
        frame, _ = hamming_decode(ordered)
        if len(frame) < 8 or not frame.startswith(b'BP1 '):
            continue
        body = frame[4:-4]
        crc_tail = int.from_bytes(frame[-4:], 'big')
        if (zlib.crc32(body) & 0xffffffff) != crc_tail:
            continue
        try:
            flag = body.decode('ascii')
        except UnicodeDecodeError:
            continue
        if re.fullmatch(r'[A-Za-z0-9_]+\{[^\r\n{}]+\}', flag):
            return flag
    return None


def demodulate_and_decode(samples, sample_rate, symbol_rate, carrier, training, lanes):
    sps = int(round(sample_rate / symbol_rate))
    if sps <= 0:
        raise ValueError('bad sample/symbol rate')
    alt10 = [1 if i % 2 == 0 else 0 for i in range(64)]
    alt01 = [1 - b for b in alt10]
    two_pi = 2.0 * math.pi

    for offset in range(sps):
        symbols = []
        for idx in range(offset, len(samples), sps):
            angle = -two_pi * carrier * idx / sample_rate
            symbols.append(samples[idx] * complex(math.cos(angle), math.sin(angle)))
        if len(symbols) <= training:
            continue

        phasor = sum(v * v for v in symbols)
        phase = 0.5 * math.atan2(phasor.imag, phasor.real)
        rot = complex(math.cos(-phase), math.sin(-phase))
        bits = [1 if (v * rot).real < 0 else 0 for v in symbols]

        m10 = sum(bits[i] == alt10[i] for i in range(64))
        m01 = sum(bits[i] == alt01[i] for i in range(64))
        if m01 > m10:
            bits = [1 - b for b in bits]
            m10 = m01
        if m10 < 60:
            continue

        flag = try_decode_symbol_bits(bits, training, lanes)
        if flag:
            return flag
    raise ValueError('flag not recovered')


def main():
    if len(sys.argv) != 2:
        print('usage: python 1.py <attachment_path>')
        sys.exit(1)
    note_text, csv_text = read_bundle(sys.argv[1])
    sample_rate, symbol_rate, carrier, training, lanes = parse_note(note_text)
    samples = read_iq(csv_text)
    flag = demodulate_and_decode(samples, sample_rate, symbol_rate, carrier, training, lanes)
    print(flag)


if __name__ == '__main__':
    main()
```
