
### 一、基础信息收集与分析
将提供的压缩包释放后，我们可以发现四个文件：一张图片、一个模型状态存档、一个 JSON 配置文件以及一个 Windows 下的可执行程序。根据题目给出的引言“细微之处见真章”，我们应当将突破口锁定在图片最低有效位（LSB）隐写、不同颜色通道间的细微区别，或者是模型文件中潜藏的附加数据上。

```text
Archive: TrainModel.zip
  Length      Name
      173     config.json
   172692     Input.png
     8178     model_state.npz
 18091520     train.exe
```

首先对图片的 RGB 通道进行观测。虽然从视觉上这是一张常规的灰度 Lena 图，但实际上三个通道的数据并不完全一致。经过对比可以发现，区别仅仅存在于极少部分像素的最低有效位（LSB）中。

```text
R/G/B LSB 置 1 数量：R=58, G=59, B=65
R-G 差值范围：[-1, 1]
G-B 差值范围：[-1, 1]
```

### 二、分析 train.exe 获取解题思路
通过逆向分析可知，`train.exe` 是一个由 Nuitka 打包而成的单文件（onefile）程序。在其资源段内，隐藏着一段以字符 `KAY` 打头、随后紧接着 zstd 压缩特征字（magic）的海量数据。将开头的 3 个字节剔除后进行解压，即可暴露出 `train.dll`、`core_secret.pyd` 等核心逻辑文件，以及 numpy 和 PIL 等环境依赖库。

```bash
RT_RCDATA: 4B 41 59 28 B5 2F FD ...
# 4B 41 59 = "KAY"
# 28 B5 2F FD = zstd frame magic
 
tail -c +4 res10.bin | zstdcat > payload.bin
```

进一步对提取出的 `core_secret.pyd` 进行探查，能够清晰地发现暴露的关键函数名称及注释说明。这为我们指明了核心步骤：需要依次从 R、G、B 这三个通道里抽取出隐藏的载荷（payload）。

```text
_extract_channel_lsb
extract_multichannel_payloads
payloads: {"R": bytes, "G": bytes, "B": bytes}
ObfuscatedPayloadProcessor
scramble_between_markers
```

### 三、解析 model_state.npz 隐藏的线索
在探查 `model_state.npz` 文件时，可以非常直观地察觉到两个特殊的键值：`base_aux_meta` 与 `base_hint_bytes`。其中 `base_aux_meta` 的值为 `b64+gz`，这无疑在暗示我们，对于 `base_hint_bytes` 中的数据，应当先执行 Base64 解码操作，随后再通过 gzip 进行解压缩。

```python
base_aux_meta  = b"b64+gz"
base_hint_bytes = b"H4sIAFPoKmkC//MMdnau9nUx1UjLSUw3DKqpAdFG7pq1AAFqTvkZAAAA"
```

完成上述解密流程后，即可获取到 Flag 的最终生成逻辑：

```text
ISCC{MD5(flag1R||flag2G)}
```

这表明，接下来的目标就是分别从红色（R）和绿色（G）通道提取出两段过渡形态的 payload，将它们合并后计算 MD5 哈希值即可。

### 四、依通道次序提取 LSB 隐藏信息
根据前文在 `core_secret.pyd` 里得到的提示，位序规则为最高有效位（MSB）优先。所以，在我们获取到各个通道的 LSB 序列后，必须按照大端模式（big-endian），将每 8 个比特位重新组合成单个字节。截取最前面的 16 个字节，就能成功拿到形如 `ISCC{...}` 的文本载荷。

```python
bits = (img[:, :, channel].reshape(-1) & 1)[:16 * 8]
raw  = np.packbits(bits, bitorder="big").tobytes()
text = raw.decode("ascii")
```

各通道数据还原情况如下：

| 颜色通道 | 提取结果 | 备注说明 |
| :---: | :--- | :--- |
| **R** | `ISCC{qhLEGHTGSQ}` | 对应 `flag1R` |
| **G** | `ISCC{eECNXdfeBt}` | 对应 `flag2G` |
| **B** | `ISCC{WHcoiqznHL}` | 为混淆视听的干扰项，按提示无需使用 |

### 五、推导最终 FLAG
依照前面获取到的 `flag1R||flag2G` 拼接规则，我们将从 R 通道和 G 通道提取出的两段完整字符串直接相连，并对其进行 MD5 加密运算。

```text
flag1R = "ISCC{qhLEGHTGSQ}"
flag2G = "ISCC{eECNXdfeBt}"
md5(flag1R + flag2G) = 5ad920ea777d9b6c24311ffd3d62ef42
```

### 附录：自动化利用（EXP）脚本与执行回显

**EXP 运行回显**
```text
[+] hint: ISCC{MD5(flag1R||flag2G)}
[+] R payload: ISCC{qhLEGHTGSQ}
[+] G payload: ISCC{eECNXdfeBt}
[+] B payload: ISCC{WHcoiqznHL}
```

**EXP 完整源码**
```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
 
from __future__ import annotations
 
import base64
import gzip
import hashlib
import zipfile
from pathlib import Path
 
import numpy as np
from PIL import Image
 
 
def bits_to_bytes_msb(bits: np.ndarray) -> bytes:
    """把 MSB 优先的 bit 序列还原为 bytes。"""
    bits = np.asarray(bits, dtype=np.uint8)
    pad = (-len(bits)) % 8
    if pad:
        bits = np.concatenate([bits, np.zeros(pad, dtype=np.uint8)])
    return np.packbits(bits, bitorder="big").tobytes()
 
 
def decode_npz_hint(npz_path: Path) -> str:
    state = np.load(npz_path, allow_pickle=True)
    hint_bytes = bytes(state["base_hint_bytes"])
    aux_meta = bytes(state["base_aux_meta"]).decode("ascii", errors="ignore")
    if aux_meta != "b64+gz":
        raise RuntimeError(f"unknown aux meta: {aux_meta!r}")
    return gzip.decompress(base64.b64decode(hint_bytes)).decode("ascii")
 
 
def extract_channel_payload(img: np.ndarray, channel: int, nbytes: int = 16) -> str:
    """按行优先顺序读取指定通道 LSB，并按 MSB 优先组字节。"""
    nbits = nbytes * 8
    flat_bits = (img[:, :, channel].reshape(-1) & 1)[:nbits]
    raw = bits_to_bytes_msb(flat_bits)
    return raw.decode("ascii")
 
 
def solve(workdir: Path) -> str:
    npz_path = workdir / "model_state.npz"
    img_path = workdir / "Input.png"
 
    hint = decode_npz_hint(npz_path)
    print("[+] hint:", hint)
 
    img = np.array(Image.open(img_path).convert("RGB"))
    payload_r = extract_channel_payload(img, 0)
    payload_g = extract_channel_payload(img, 1)
    payload_b = extract_channel_payload(img, 2)
 
    print("[+] R payload:", payload_r)
    print("[+] G payload:", payload_g)
    print("[+] B payload:", payload_b)
 
    digest = hashlib.md5((payload_r + payload_g).encode("ascii")).hexdigest()
    flag = f"ISCC{{{digest}}}"
    print("[+] flag:", flag)
    return flag
 
 
if __name__ == "__main__":
    root = Path(__file__).resolve().parent
    zip_path = root / "TrainModel.zip"
    out_dir = root / "TrainModel_extracted_by_exp"
    if zip_path.exists() and not (out_dir / "Input.png").exists():
        out_dir.mkdir(exist_ok=True)
        with zipfile.ZipFile(zip_path, "r") as zf:
            zf.extractall(out_dir)
    else:
        out_dir = Path("/mnt/data/train_extracted")
    solve(out_dir)
```
