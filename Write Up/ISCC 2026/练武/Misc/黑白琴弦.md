
## 一、外层压缩包分析

题目附件为：

```text
attachment-18.rar
```

先使用 `file` 或 7-Zip / WinRAR 识别文件格式：

```bash
file attachment-18.rar
```

可以确认这是一个 RAR 压缩包。

解压外层压缩包：

```bash
7z x attachment-18.rar -oouter
```

解压后可以看到：

```text
barcode_*.png
score.rar
```

其中有 21 张条形码图片，以及一个二层压缩包 `score.rar`。

此时可以判断：

1. 21 张 `barcode_*.png` 是第一层线索。
2. `score.rar` 很可能是加密压缩包，需要从图片中提取密码。
3. 题目名为“黑白琴弦”，后续可能还会出现乐谱、钢琴、五线谱相关内容。

---

## 二、分析 barcode 图片

### 1. 直接识别条形码

对 21 张 PNG 图片进行分析，发现它们都是 Code128 条形码。

可以尝试使用工具识别：

```bash
zbarimg barcode_*.png
```

或者使用 Python 自己解析 Code128。

但是本题的关键点在于：每张 PNG 中不止有条形码数据，还存在其他隐藏信息。

---

### 2. 检查 PNG 元数据

继续检查 PNG 的文本块和注释字段：

```bash
exiftool barcode_001.png
strings barcode_001.png
```

部分图片中可以在 PNG Comment 中发现可疑字符串。

这些字符串格式类似：

```text
A1fc97fA
BxxxxxxB
...
U1fd38bU
```

特征是：

```text
字母 + 6 位十六进制 + 相同字母
```

例如：

```text
A1fc97fA
```

其中：

- `A` 是编号
- `1fc97f` 是 6 位 hex 数据
- 结尾的 `A` 用于确认这一行属于 A

---

### 3. 检查第 0 行像素 LSB

除了条形码和 PNG Comment 外，部分信息还隐藏在图片第 0 行像素的红色通道最低位中。

提取方式如下：

```python
from PIL import Image

def read_first_row_red_lsb(path):
    img = Image.open(path).convert("RGBA")
    w, h = img.size

    bits = []
    for x in range(w):
        r, g, b, a = img.getpixel((x, 0))
        bits.append(r & 1)

    data = bytearray()
    for i in range(0, len(bits) // 8 * 8, 8):
        v = 0
        for bit in bits[i:i + 8]:
            v = (v << 1) | bit
        data.append(v)

    if not data:
        return ""

    length = data[0]
    return bytes(data[1:1 + length]).decode(errors="ignore")
```

这样可以从不同来源得到 21 组有效数据：

```text
AxxxxxxA
BxxxxxxB
CxxxxxxC
...
UxxxxxxU
```

---

## 三、重建 21 × 21 二维码

### 1. 按字母顺序排序

因为数据编号从 `A` 到 `U`，一共 21 个字母，所以可以判断它们对应二维码的 21 行。

每一行的数据为 6 位十六进制：

```text
6 hex = 24 bit
```

但本题实际使用每组数据的低 21 位作为二维码的一行。

处理逻辑如下：

```python
bits24 = bin(int(hex6, 16))[2:].zfill(24)
row_bits = bits24[-21:]
```

将 `A` 到 `U` 依次作为二维码的第 1 行到第 21 行，即可得到一个：

```text
21 × 21 QR Code
```

21 × 21 正好是 QR Code Version 1 的大小。

---

### 2. 生成二维码图片

示例代码：

```python
from PIL import Image

def save_qr(rows, out="qr.png", scale=10, quiet=4):
    n = 21
    img = Image.new("L", ((n + quiet * 2) * scale, (n + quiet * 2) * scale), 255)

    for y, row in enumerate(rows):
        for x, bit in enumerate(row):
            if bit == "1":
                for yy in range((y + quiet) * scale, (y + quiet + 1) * scale):
                    for xx in range((x + quiet) * scale, (x + quiet + 1) * scale):
                        img.putpixel((xx, yy), 0)

    img.save(out)
```

生成二维码后识别：

```bash
zbarimg qr.png
```

或者使用 OpenCV / pyzbar 识别。

识别结果为：

```text
OKwbosL4uQZEYfh
```

这就是二层压缩包 `score.rar` 的密码，同时也是最终 flag 的前半部分 key。

---

## 四、解开二层压缩包 score.rar

使用二维码得到的字符串作为密码：

```bash
7z x score.rar -pWgs6GdF731LwhQz -oscore_out
```

成功解压后得到：

```text
music score.pdf
```

说明二维码结果判断正确。

## 五、分析 PDF 文件

对 PDF 先做常规检查：

```bash
strings "music score.pdf"
pdftotext "music score.pdf" -
binwalk "music score.pdf"
```

同时也可以解析 PDF 中的压缩流。

在 PDF 隐藏文本中发现：

```text
KEY+bnVtYmVyYWJvdmVsaW5l3
```

将 `KEY+` 后面的主要部分进行 Base64 解码：

```bash
echo bnVtYmVyYWJvdmVsaW5l | base64 -d
```

得到：

```text
numberaboveline
```

因此完整提示可以理解为：

```text
KEY + number above line 3
```

也就是：

```text
二维码得到的 key + 第 3 行上方的数字
```

其中 `KEY` 对应前面二维码解出的：

```text
OKwbosL4uQZEYfh
```

## 六、结合乐谱内容提取数字

将 PDF 渲染成图片：

```bash
pdftoppm -png "music score.pdf" page
```

或者：

```bash
python -m fitz draw -o page.png "music score.pdf"
```

打开渲染出的乐谱图片，观察第 3 行乐谱上方的数字。

注意这里的 `number above line 3` 不是指第三行左侧的小节号，而是指第 3 行乐谱上方标注在音符上面的数字串。

第 3 行上方可以看到数字：

```text
4 1 5 1 5 1 5
```

连起来得到：

```text
4151515
```

---

## 七、拼接最终 flag

根据 PDF 隐藏提示：

```text
KEY + number above line 3
```

其中：

```text
KEY = OKwbosL4uQZEYfh
number above line 3 = 4151515
```

所以最终内容为：

```text
OKwbosL4uQZEYfh4151515
```

按照题目要求包上 `ISCC{}`：

```text
ISCC{OKwbosL4uQZEYfh4151515}
```

## 八、通用自动化脚本

下面脚本完成主要自动化流程：

1. 解压外层 RAR；
2. 遍历 barcode PNG；
3. 从 PNG Comment 和第 0 行红色通道 LSB 提取数据；
4. 按 A 到 U 排序；
5. 重建 QR；
6. 输出二维码 key；
7. 提示人工查看 PDF 第 3 行上方数字；
8. 拼接最终 flag。

> 说明：Code128 解码可用 `zbarimg`、`pyzbar` 或自写解析器完成。如果环境缺少条形码库，也可以只依赖 PNG Comment 和 LSB，因为本题有效数据分散在多个位置。

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-

import os
import re
import shutil
import subprocess
from pathlib import Path
from PIL import Image

def run(cmd):
    print("[cmd]", " ".join(map(str, cmd)))
    p = subprocess.run(cmd, stdout=subprocess.PIPE, stderr=subprocess.STDOUT, text=True)
    if p.returncode != 0:
        raise RuntimeError(p.stdout)
    return p.stdout

def extract_archive(archive, outdir, password=None):
    archive = Path(archive)
    outdir = Path(outdir)
    outdir.mkdir(parents=True, exist_ok=True)

    attempts = []

    cmd_7z = ["7z", "x", "-y", f"-o{outdir}"]
    if password:
        cmd_7z.append(f"-p{password}")
    cmd_7z.append(str(archive))
    attempts.append(cmd_7z)

    if archive.suffix.lower() == ".rar" and shutil.which("unrar"):
        cmd_unrar = ["unrar", "x", "-o+"]
        cmd_unrar.append(f"-p{password}" if password else "-p-")
        cmd_unrar.extend([str(archive), str(outdir) + os.sep])
        attempts.append(cmd_unrar)

    errors = []
    for cmd in attempts:
        print("[cmd]", " ".join(map(str, cmd)))
        p = subprocess.run(cmd, stdout=subprocess.PIPE, stderr=subprocess.STDOUT, text=True)
        if p.returncode == 0:
            return p.stdout
        errors.append(p.stdout)
        print("[!] 解压失败，尝试下一个后端")

    raise RuntimeError("\n\n".join(errors))

def read_png_comment(path):
    img = Image.open(path)
    return img.info.get("Comment") or img.info.get("comment") or ""

def read_first_row_red_lsb(path):
    img = Image.open(path).convert("RGBA")
    w, h = img.size

    bits = []
    for x in range(w):
        r, g, b, a = img.getpixel((x, 0))
        bits.append(r & 1)

    raw = bytearray()
    for i in range(0, len(bits) // 8 * 8, 8):
        v = 0
        for bit in bits[i:i + 8]:
            v = (v << 1) | bit
        raw.append(v)

    if not raw:
        return ""

    length = raw[0]
    try:
        return bytes(raw[1:1 + length]).decode("ascii")
    except Exception:
        return ""

def read_barcode_text(path):
    try:
        out = run(["zbarimg", "--raw", str(path)])
        line = out.strip().splitlines()
        return line[0].strip() if line else ""
    except Exception:
        return ""

def collect_rows(img_dir):
    pattern = re.compile(r"^([A-U])([0-9a-fA-F]{6})\1$")
    rows = {}

    for path in sorted(Path(img_dir).glob("*.png")):
        candidates = []

        comment = read_png_comment(path)
        if comment:
            candidates.append(("comment", comment))

        lsb = read_first_row_red_lsb(path)
        if lsb:
            candidates.append(("lsb", lsb))

        barcode = read_barcode_text(path)
        if barcode:
            candidates.append(("barcode", barcode))

        for source, text in candidates:
            m = pattern.match(text.strip())
            if not m:
                continue

            letter = m.group(1)
            hex6 = m.group(2).lower()

            rows[letter] = hex6
            print(f"[+] {path.name}: {source} -> {letter}{hex6}{letter}")

    missing = [chr(ord("A") + i) for i in range(21) if chr(ord("A") + i) not in rows]
    if missing:
        print("[!] 缺少部分行：", missing)
        print("[!] 可以继续补充 Code128 条形码识别结果。")

    return rows

def build_qr(rows, out="rebuilt_qr.png", scale=10, quiet=4):
    letters = [chr(ord("A") + i) for i in range(21)]
    grid = []

    for ch in letters:
        hex6 = rows[ch]
        bits24 = bin(int(hex6, 16))[2:].zfill(24)
        row = bits24[-21:]
        grid.append(row)

    n = 21
    img = Image.new("L", ((n + quiet * 2) * scale, (n + quiet * 2) * scale), 255)

    for y, row in enumerate(grid):
        for x, bit in enumerate(row):
            if bit == "1":
                for yy in range((y + quiet) * scale, (y + quiet + 1) * scale):
                    for xx in range((x + quiet) * scale, (x + quiet + 1) * scale):
                        img.putpixel((xx, yy), 0)

    img.save(out)
    print("[+] QR saved:", out)
    return out

def decode_qr_with_zbar(qr_path):
    try:
        out = run(["zbarimg", "--raw", qr_path])
        return out.strip().splitlines()[0]
    except Exception as e:
        print("[!] zbarimg 识别失败，请手动扫描 rebuilt_qr.png")
        print(e)
        return None

def main():
    rar = Path("attachment-18.rar")
    work = Path("work_black_white_piano")

    if work.exists():
        shutil.rmtree(work)

    outer = work / "outer"
    score_out = work / "score_out"

    print("[*] 解压外层压缩包")
    extract_archive(rar, outer)

    print("[*] 收集 A-U 的 6 位 hex 数据")
    rows = collect_rows(outer)

    if len(rows) != 21:
        raise RuntimeError("没有收集到完整 21 行数据，需要补充条形码识别结果。")

    print("[*] 重建二维码")
    qr = build_qr(rows, work / "rebuilt_qr.png")

    print("[*] 识别二维码")
    key = decode_qr_with_zbar(str(qr))
    if not key:
        key = input("请输入手动识别二维码得到的 key: ").strip()

    print("[+] QR key =", key)

    score_rars = list(outer.rglob("score.rar"))
    if not score_rars:
        raise RuntimeError("未找到 score.rar")

    print("[*] 解压 score.rar")
    extract_archive(score_rars[0], score_out, password=key)

    pdfs = list(score_out.rglob("*.pdf"))
    print("[+] 解出的 PDF：")
    for pdf in pdfs:
        print("   ", pdf)

    print()
    print("[*] PDF 隐藏提示为 KEY+bnVtYmVyYWJvdmVsaW5l3")
    print("[*] Base64 解码得到 numberaboveline，因此需要查看乐谱第 3 行上方数字。")
    print("[*] 本题第 3 行上方数字串为：4151515")

    suffix = "4151515"
    flag = f"ISCC{{{key}{suffix}}}"

    print()
    print("========== FLAG ==========")
    print(flag)
    print("==========================")

if __name__ == "__main__":
    main()
```
