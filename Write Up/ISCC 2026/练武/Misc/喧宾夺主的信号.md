
## 解题过程

### 步骤 1 - 区分噪声与真实信号

**关键观察**

先做最小侦察，不要一开始就深挖 1500 个组播包。协议层统计能直接把主线分成两部分：一条是固定节奏、固定目的地址的 SSDP/组播噪声；另一条是数量极少的 TCP/HTTP 小会话。真正的控制指令通常藏在后者，前者只负责投放口令或钥匙。

**关键命令**

```bash
tshark -r attachment-17.pcapng -q -z io,phs

tshark -r attachment-17.pcapng -q -z conv,ip
```

**关键输出**

```text
Protocol Hierarchy Statistics
frame 1509
  udp 1500
  tcp 9
    http 1

IPv4 Conversations
192.168.1.100 <-> 239.255.255.250    1500 frames   84 kB
192.168.1.100 <-> 45.78.1.1             9 frames  1148 bytes
```

**补充说明**

这一步的目标不是读懂所有组播内容，而是先确认“真正值得跟踪的极少数会话”在哪里。只要把噪声层和指令层分离，后面的分析量会立刻收缩。

### 步骤 2 - 从组播噪声中恢复 ZIP 口令

**关键观察**

组播 payload 看起来像随机噪声，但其中夹着可打印的 Base64 片段。对整份抓包做字符串筛选后，再按“可 Base64 解码且解码后仍是可打印 ASCII”的规则筛选，就能得到真正的口令候选。动态附件中这个字符串会变化，但恢复规则保持不变。

**关键命令**

```bash
strings -a attachment-17.pcapng | rg '^[A-Za-z0-9+/=]{20,}$' | sort | uniq -c | sort -nr | head

echo 'U2hlbmdEb25nSmlYaUAzNi0xLTY=' | base64 -d
```

**关键输出**

```text
4 U2hlbmdEb25nSmlYaUAzNi0xLTY=
ShengDongJiXi@36-1-6
```

**补充说明**

样例附件中的口令是 `ShengDongJiXi@36-1-6`，显然和题面“三十六计”提示相呼应。换成其他动态附件时，具体字符串会不同，但同样会以可打印 Base64 片段的形式藏在组播噪声层里。

### 步骤 3 - 从 HTTP JSON 中提取 AES ZIP 并解出 PNG

**关键观察**

沿着那条极少量的 TCP/HTTP 会话继续看，可以直接看到 `POST /command`，请求体中包含一个 JSON 字段 `instruction`。它是 Base64 编码的 ZIP 包，解码后会发现 ZIP 使用了 AES 加密。此时把上一步从组播里恢复出的口令喂给 ZIP 解压工具即可得到内层 PNG。

**关键命令**

```bash
tshark -r attachment-17.pcapng -q -z follow,tcp,ascii,0

tshark -r attachment-17.pcapng -Y 'http.request.method=="POST"' -T fields -e http.file_data | xxd -r -p > body.json
jq -r '.instruction' body.json | base64 -d > instruction.zip
7z x -p'<recovered-password>' instruction.zip
```

**关键输出**

```text
POST /command HTTP/1.1
Host: 45.78.1.1
Content-Type: application/json
{"instruction": "<base64-zip>", "note": "This is the real command."}

instruction.zip: Zip archive data, AES Encrypted
Extracted: image.png (100x100 PNG)
```

**补充说明**

这张图本身非常小，颜色数量也极少。对样例附件做统计时，只能看到 8 个近似颜色值，说明 RGB 通道只在最低位上发生了轻微抖动，这正是继续走 LSB 位流提取的强信号。

### 步骤 4 - 按像素顺序提取 RGB-LSB 位流并恢复动态 flag

**关键观察**

最终层不需要猜测复杂隐写算法。直接按像素顺序读取 PNG 的 RGB 三通道最低位，按 `R,G,B` 顺序拼接 bit，再每 8 位按高位在前打包成字节流即可。样例附件在字节流开头直接出现 `ISCC{...}`。不同动态附件会生成不同 flag，但这一步的读取顺序与编码规则不变。

## 附录1：EXP

```python
#!/usr/bin/env python3
"""
Generic solver for the dynamic PCAP challenge "喧宾夺主的信号".

Workflow:
1. Find the encrypted ZIP stored in the HTTP JSON field "instruction".
2. Recover candidate passwords from repeated printable Base64 strings hidden
   inside the multicast UDP noise.
3. Try those passwords against the ZIP until extraction succeeds.
4. Read the extracted PNG, pack RGB LSBs in pixel order, and recover the flag.

Notes:
- The password string and final flag both vary per attachment.
- This EXP avoids hardcoded secrets and derives them from the current capture.
- AES ZIP extraction uses `7z` when available, and falls back to `pyzipper`.
- PNG decoding requires Pillow.
"""

from __future__ import annotations

import argparse
import base64
import binascii
import re
import shutil
import string
import subprocess
import sys
import tempfile
from collections import Counter
from pathlib import Path


FLAG_PATTERNS = [
    rb"ISCC\{[^}\r\n]+\}",
    rb"[A-Za-z]{2,16}\{[^}\r\n]+\}",
]


def read_blob(path: Path) -> bytes:
    return path.read_bytes()


def extract_instruction_zip(blob: bytes) -> bytes:
    match = re.search(rb'"instruction"\s*:\s*"([A-Za-z0-9+/=]+)"', blob)
    if not match:
        raise ValueError("instruction field not found in capture")
    return base64.b64decode(match.group(1), validate=True)


def looks_printable_password(data: bytes) -> bool:
    if not (6 <= len(data) <= 96):
        return False
    if any(ch not in bytes(string.printable, "ascii") for ch in data):
        return False
    decoded = data.decode("ascii", errors="ignore")
    if len(decoded.strip()) != len(decoded):
        return False
    if sum(ch.isalnum() for ch in decoded) < max(4, len(decoded) // 3):
        return False
    return any(ch in decoded for ch in "@-_!#$%^&*")


def password_candidates(blob: bytes) -> list[tuple[str, int]]:
    counter: Counter[str] = Counter()
    for token in re.findall(rb"(?<![A-Za-z0-9+/=])[A-Za-z0-9+/=]{20,128}(?![A-Za-z0-9+/=])", blob):
        try:
            decoded = base64.b64decode(token, validate=True)
        except binascii.Error:
            continue
        if looks_printable_password(decoded):
            counter[decoded.decode("ascii")] += 1
    return sorted(counter.items(), key=lambda item: (-item[1], -len(item[0]), item[0]))


def extract_with_7z(zip_path: Path, password: str, out_dir: Path) -> Path | None:
    if shutil.which("7z") is None:
        return None
    shutil.rmtree(out_dir, ignore_errors=True)
    out_dir.mkdir(parents=True, exist_ok=True)
    result = subprocess.run(
        ["7z", "x", f"-p{password}", "-y", f"-o{out_dir}", str(zip_path)],
        stdout=subprocess.PIPE,
        stderr=subprocess.STDOUT,
        text=True,
    )
    if result.returncode != 0:
        return None
    files = [path for path in out_dir.rglob("*") if path.is_file()]
    return files[0] if files else None


def extract_with_pyzipper(zip_path: Path, password: str, out_dir: Path) -> Path | None:
    try:
        import pyzipper
    except ImportError:
        return None

    shutil.rmtree(out_dir, ignore_errors=True)
    out_dir.mkdir(parents=True, exist_ok=True)
    try:
        with pyzipper.AESZipFile(zip_path) as zf:
            zf.pwd = password.encode()
            members = [name for name in zf.namelist() if not name.endswith("/")]
            if not members:
                return None
            member = members[0]
            zf.extract(member, path=out_dir)
            return out_dir / member
    except Exception:
        return None


def extract_zip_member(zip_bytes: bytes, passwords: list[str], keep_dir: Path | None = None) -> tuple[Path, str]:
    if not passwords:
        raise ValueError("no printable Base64 password candidates recovered from multicast noise")

    if keep_dir is None:
        temp_root = Path(tempfile.mkdtemp(prefix="iscc-signal-"))
    else:
        keep_dir.mkdir(parents=True, exist_ok=True)
        temp_root = keep_dir

    zip_path = temp_root / "instruction.zip"
    zip_path.write_bytes(zip_bytes)

    for password in passwords:
        out_dir = temp_root / "unzipped"
        member = extract_with_7z(zip_path, password, out_dir)
        if member is None:
            member = extract_with_pyzipper(zip_path, password, out_dir)
        if member is not None and member.exists():
            return member, password

    raise ValueError("failed to decrypt ZIP with recovered candidates; try --password to override")


def rgb_lsb_bytes(image_path: Path) -> bytes:
    try:
        from PIL import Image
    except ImportError as exc:
        raise RuntimeError("Pillow is required: pip install pillow") from exc

    with Image.open(image_path) as img:
        rgb = img.convert("RGB")
        width, height = rgb.size
        pixels = rgb.load()
        bits: list[int] = []
        for y in range(height):
            for x in range(width):
                pixel = pixels[x, y]
                bits.extend((pixel[0] & 1, pixel[1] & 1, pixel[2] & 1))

    out = bytearray()
    for i in range(0, len(bits) // 8 * 8, 8):
        value = 0
        for bit in bits[i : i + 8]:
            value = (value << 1) | bit
        out.append(value)
    return bytes(out)


def printable_runs(data: bytes, min_len: int = 12) -> list[str]:
    runs: list[str] = []
    current: list[str] = []
    for byte in data:
        if 32 <= byte <= 126:
            current.append(chr(byte))
        else:
            if len(current) >= min_len:
                runs.append("".join(current))
            current = []
    if len(current) >= min_len:
        runs.append("".join(current))
    return runs


def recover_flag(data: bytes) -> str:
    for pattern in FLAG_PATTERNS:
        match = re.search(pattern, data)
        if match:
            return match.group(0).decode("ascii", errors="ignore")

    runs = printable_runs(data, min_len=20)
    if runs:
        return runs[0]
    raise ValueError("no flag-like printable run found in RGB LSB payload")


def parse_args() -> argparse.Namespace:
    parser = argparse.ArgumentParser(description="Recover the dynamic flag from the PCAP attachment.")
    parser.add_argument("pcap", help="Path to the challenge capture")
    parser.add_argument("--password", help="Optional manual ZIP password override")
    parser.add_argument("--keep-dir", help="Directory used to keep intermediate ZIP and extracted files")
    return parser.parse_args()


def main() -> int:
    args = parse_args()
    pcap_path = Path(args.pcap).expanduser().resolve()
    blob = read_blob(pcap_path)
    zip_bytes = extract_instruction_zip(blob)

    recovered = password_candidates(blob)
    password_list = [args.password] if args.password else [value for value, _count in recovered]
    if not password_list:
        print("[-] no password candidates found", file=sys.stderr)
        return 1

    print(f"[*] capture        = {pcap_path}")
    print(f"[*] zip bytes      = {len(zip_bytes)}")
    if recovered:
        print("[*] candidate passwords recovered from UDP noise:")
        for value, count in recovered[:8]:
            print(f"    - {value} (seen {count}x)")
    elif args.password:
        print(f"[*] using manual password override: {args.password}")

    keep_dir = Path(args.keep_dir).expanduser().resolve() if args.keep_dir else None
    image_path, used_password = extract_zip_member(zip_bytes, password_list, keep_dir=keep_dir)
    print(f"[*] selected password = {used_password}")
    print(f"[*] extracted member  = {image_path}")

    payload = rgb_lsb_bytes(image_path)
    flag = recover_flag(payload)
    print(f"[+] recovered flag = {flag}")
    return 0


if __name__ == "__main__":
    raise SystemExit(main())
```
