
## 分析过程

### 1. Java 层只负责串 JNI

反编译后可以看到 `MainActivity` 暴露了这几个 native 方法：

- `nativeInit`
- `nativeLogin`
- `nativeVerifyChallenge1`
- `nativeVerifyChallenge2`
- `nativeGetIntermediateToken`
- `nativeVerifyChallenge3WithToken`
- `nativeGenerateFlag`

核心逻辑都在 `libzkpcore.so`。

### 2. nativeInit 只是在做初始化和全局状态准备

`nativeInit` 主要完成三件事：

1. 分配全局 `Auth` 和 `ChallengeState`
2. 写入初始化标记
3. 落一个 `hint.txt`

题面里提到的 `device_secret / Blum square root modulo N` 只体现在 hint 文字上，真正决定第三关的是 native 里的固定计算逻辑，不需要真的去做 Blum 开方。

### 3. 登录凭据可以从 rodata 里恢复

用户名不是明文直接给出，而是通过 `k3y_j4va_k3y!!!!` 和其后面的 mask 异或得到：

```python
marker = b"k3y_j4va_k3y!!!!"
mask = rodata[off + len(marker): off + len(marker) + 10]
username = bytes(marker[i] ^ mask[i] for i in range(10)).decode()
```

当前附件恢复结果：

```text
zkp_master
```

密码由三段字符串拼接：

```text
Zkp_ + M_as + t3er
```

得到：

```text
Zkp_M_ast3er
```

### 4. 第一关是一个可逆的 Feistel 风格校验

第一关内部仍然是固定模板，但这次最关键的几个参数不该写死，而应该直接从 `CHECK_CH1` 的机器码里提取：

- LFSR seed
- 4 个 round key 常量
- 最终比较目标值

当前附件里提取出的参数分别是：

```text
seed   = 0x2161
const0 = 0xd5d86bc8
const1 = 0xee944448
const2 = 0xedca8a96
const3 = 0x3b4c2f80
target = 0x66ab7b49
```

拿到这些参数后，直接对这段 Feistel 风格变换做逆运算，就能恢复第一关输入。脚本里随后还会把结果重新交给真实 `CHECK_CH1` 复验，确保不是静态推导写偏。

当前附件第一关结果：

```text
5827A462
```

### 5. 第二关直接让 native 自己生成

第二关没必要纯静态手推，直接调用 native 里的生成函数 `FN_MAKE_CH2(0x4034)` 即可得到真实答案，再交给 `CHECK_CH2` 校验。

当前附件第二关结果：

```text
5c44540653d49a8066ac3e1ea04da5d1
```

第二关通过后，native 会把中间 token 写进全局缓冲区。

当前附件 token：

```text
1dae0a8f1f76671bf03701dfc420d400
```

### 6. 第三关的关键是附件内的 64 位异或常量

第三关并不是题面提示的那套“Blum square root modulo N”，真实逻辑是：

1. 取 `token` 前 8 字节
2. 按 native 的方式打包成 64 位整数
3. 与一个 64 位常量异或
4. 对结果做 `sha256`
5. 取十六进制前 16 个字符作为第三关答案

也就是：

```python
packed = pack(token[:8])
mixed = packed ^ xor_const
challenge3 = sha256(mixed.to_bytes(8, "little")).hex()[:16]
```

这里最关键的点是：`xor_const` 不是固定值，应该从当前附件的 `CHECK_CH3` 机器码里动态提取。

当前附件提取出的常量是：

```text
0xe445f13ff7329ca2
```

因此第三关答案为：

```text
d725f8d3e451aa97
```

### 7. 最终 flag 通过 native 生成

最后调用 native 里的 flag 生成函数即可。

需要注意的是，JNI wrapper 会维护 `ChallengeState` 中的计数器和三个位标记：

- `state[0:4]`：已通过关卡计数
- `state[4]`：challenge1 已通过
- `state[5]`：challenge2 已通过
- `state[6]`：challenge3 已通过

如果只调 core、不补这些状态位，`generateFlag` 会失败。

补齐状态后，当前附件得到：

```text
ISCC{30942ba24e3e64085d307545741e58f88c00f7be07b5a158}
```

## 通用脚本

下面这份脚本是按“同一 native 模板，附件内嵌数据可变化”的思路写的：

- 不写死当前附件的 `challenge1` 参数
- 不写死当前附件的 `challenge2`
- 不写死当前附件的 `token`
- 不写死当前附件的 `challenge3 xor 常量`
- 不写死当前附件的 `flag`

依赖：

```bash
pip install pyelftools unicorn
```

运行方式：

```bash
python3 attachment-25-generic-exp.py /path/to/attachment-25.apk
```

也可以在 APK 所在目录直接运行：

```bash
python3 attachment-25-generic-exp.py
```

RET_ADDR = 0xDEAD0000
HEAP_BASE = 0x1000000
STACK_BASE = 0x2000000
STACK_SIZE = 0x100000

# Same native template, per-attachment embedded data changes.
FN_NATIVE_INIT = 0x1DE4
FN_ALLOC_AUTH = 0x297C
FN_ALLOC_STATE = 0x2E84
FN_LOGIN_CORE = 0x2A0C
FN_CHECK_CH1_CORE = 0x336C
FN_MAKE_CH2 = 0x4034
FN_CHECK_CH2_CORE = 0x3C4C
FN_CHECK_CH3_CORE = 0x43D0
FN_GEN_FLAG_CORE = 0x2F14

# Offsets inside JNI wrappers / core for the current template.
OFF_INIT_ADRP_AUTH = 0x78
OFF_INIT_ADRP_STATE = 0xB0
OFF_INIT_ADRP_READY = 0x1C0
OFF_CH3_TOKEN_ADRP = 0x50
OFF_CH1_SEED_MOV = 0x118
OFF_CH1_CONST_MOVS = (0x188, 0x1A4, 0x1C0, 0x1DC)
OFF_CH1_TARGET_MOV = 0x298

CH1_SBOX = [0xE, 4, 0xD, 1, 2, 0xF, 0xB, 8, 3, 0xA, 6, 0xC, 5, 9, 0, 7]
CH1_LFSR_POLY = 0xB400


def align_down(value: int, alignment: int = 0x1000) -> int:
    return value & ~(alignment - 1)


def align_up(value: int, alignment: int = 0x1000) -> int:
    return (value + alignment - 1) & ~(alignment - 1)


def sha256(data: bytes) -> bytes:
    return hashlib.sha256(data).digest()


def first_apk() -> Path:

    if not apks:
        raise SystemExit("No APK found in current directory. Pass the APK path explicitly.")
    return apks[0]


def read_so_from_apk(apk_path: Path) -> tuple[bytes, str]:

        candidates = [name for name in zf.namelist() if name.endswith("/libzkpcore.so")]
        if not candidates:

        preferred = "lib/arm64-v8a/libzkpcore.so"
        so_name = preferred if preferred in candidates else candidates[0]

class ELFFileBytes:
    def __init__(self, data: bytes):
        from io import BytesIO

        self.bio = BytesIO(data)
        self.elf = None

    def __enter__(self):
        self.elf = ELFFile(self.bio)
        return self.elf

    def __exit__(self, exc_type, exc, tb):
        self.bio.close()


def extract_rodata(so_bytes: bytes) -> bytes:
    with ELFFileBytes(so_bytes) as elf:
        section = elf.get_section_by_name(".rodata")
        if section is None:
            raise SystemExit(".rodata not found in libzkpcore.so.")
        return section.data()


def read_u32_le(blob: bytes, offset: int) -> int:
    return struct.unpack_from("<I", blob, offset)[0]


def decode_adrp(instr: int, pc: int) -> int:
    immlo = (instr >> 29) & 0x3
    immhi = (instr >> 5) & 0x7FFFF
    imm = (immhi << 2) | immlo
    if imm & (1 << 20):
        imm -= 1 << 21
    return (pc & ~0xFFF) + (imm << 12)


def decode_add_imm(instr: int) -> int:
    imm12 = (instr >> 10) & 0xFFF
    shift = (instr >> 22) & 0x1
    return imm12 << (12 if shift else 0)


def is_movz_w(instr: int) -> bool:
    return (instr & 0x7F800000) == 0x52800000


def is_movk_w(instr: int) -> bool:
    return (instr & 0x7F800000) == 0x72800000


def decode_mov_w(instr: int) -> int:
    imm16 = (instr >> 5) & 0xFFFF
    hw = (instr >> 21) & 0x3
    return imm16 << (hw * 16)


def recover_username(rodata: bytes) -> str:
    marker = b"k3y_j4va_k3y!!!!"
    off = rodata.find(marker)
    if off == -1:
        return "zkp_master"
    mask = rodata[off + len(marker) : off + len(marker) + 10]
    return bytes(marker[i] ^ mask[i] for i in range(10)).decode()


def recover_password(rodata: bytes) -> str:
    parts = []
    for item in (b"Zkp_", b"M_as", b"t3er"):
        if rodata.find(item) == -1:
            return "Zkp_M_ast3er"
        parts.append(item)
    return b"".join(parts).decode()


def challenge1_hex_from_params(
    target: int,
    consts: tuple[int, int, int, int],
    lfsr_seed: int = 0x2161,
    sbox: list[int] | None = None,
) -> str:
    def lfsr16(x: int) -> int:
        bit = x & 1
        x >>= 1
        if bit:
            x ^= CH1_LFSR_POLY
        return x & 0xFFFF

    def sub_nibbles(x: int) -> int:
        sb = CH1_SBOX if sbox is None else sbox
        out = 0
        for i in range(8):
            out |= sb[(x >> (i * 4)) & 0xF] << (i * 4)
        return out & 0xFFFFFFFF

    def round_f(left: int, key: int) -> int:
        value = sub_nibbles(((left << 16) | left) ^ key)
        return (value ^ (value >> 16)) & 0xFFFF

    rnd = []
    seed = lfsr_seed & 0xFFFF
    for _ in range(8):
        seed = lfsr16(seed)
        rnd.append(seed)

    keys = [((rnd[2 * i] << 16) | rnd[2 * i + 1]) ^ consts[i] for i in range(4)]

    left, right = (target >> 16) & 0xFFFF, target & 0xFFFF
    for key in reversed(keys):
        old_left = right
        old_right = left ^ round_f(old_left, key)
        left, right = old_left, old_right
    return f"{(right << 16) | left:08X}"


def recover_challenge1_params(so_bytes: bytes) -> dict[str, object]:
    with ELFFileBytes(so_bytes) as elf:
        text = elf.get_section_by_name(".text")
        if text is None:
            raise SystemExit(".text not found in libzkpcore.so.")
        text_addr = text["sh_addr"]
        text_data = text.data()

    def word_at(addr: int) -> int:
        return read_u32_le(text_data, addr - text_addr)

    seed_word = word_at(FN_CHECK_CH1_CORE + OFF_CH1_SEED_MOV)
    if not is_movz_w(seed_word):
        raise RuntimeError("challenge1 seed movz not found at expected offset")
    seed = decode_mov_w(seed_word) & 0xFFFF

    consts: list[int] = []
    for offset in OFF_CH1_CONST_MOVS:
        movz = word_at(FN_CHECK_CH1_CORE + offset)
        movk = word_at(FN_CHECK_CH1_CORE + offset + 4)
        if not is_movz_w(movz) or not is_movk_w(movk):
            raise RuntimeError(f"challenge1 round constant not found at offset {offset:#x}")
        consts.append((decode_mov_w(movz) | decode_mov_w(movk)) & 0xFFFFFFFF)

    target_movz = word_at(FN_CHECK_CH1_CORE + OFF_CH1_TARGET_MOV)
    target_movk = word_at(FN_CHECK_CH1_CORE + OFF_CH1_TARGET_MOV + 4)
    if not is_movz_w(target_movz) or not is_movk_w(target_movk):
        raise RuntimeError("challenge1 target constant not found at expected offset")
    target = (decode_mov_w(target_movz) | decode_mov_w(target_movk)) & 0xFFFFFFFF

    return {"seed": seed, "consts": tuple(consts), "target": target}


def recover_movewide_constants(code: bytes) -> list[int]:
    words = [read_u32_le(code, offset) for offset in range(0, len(code) - 3, 4)]
    found: list[int] = []
    i = 0
    while i < len(words):
        word = words[i]
        kind = (word >> 23) & 0x1FF
        if kind != 0x1A5:
            i += 1
            continue
        rd = word & 0x1F
        hw = (word >> 21) & 0x3
        value = ((word >> 5) & 0xFFFF) << (hw * 16)
        used = {hw}
        j = i + 1
        while j < len(words):
            word2 = words[j]
            kind2 = (word2 >> 23) & 0x1FF
            if kind2 != 0x1E5 or (word2 & 0x1F) != rd:
                break
            hw2 = (word2 >> 21) & 0x3
            value &= ~(0xFFFF << (hw2 * 16))
            value |= ((word2 >> 5) & 0xFFFF) << (hw2 * 16)
            used.add(hw2)
            j += 1
        if len(used) >= 2:
            found.append(value & 0xFFFFFFFFFFFFFFFF)
        i = j
    return found


def recover_challenge3_constant(so_bytes: bytes) -> int:
    with ELFFileBytes(so_bytes) as elf:
        text = elf.get_section_by_name(".text")
        if text is None:
            raise SystemExit(".text not found in libzkpcore.so.")
        text_addr = text["sh_addr"]
        off = FN_CHECK_CH3_CORE - text_addr
        code = text.data()[off : off + 0x180]
    constants = recover_movewide_constants(code)
    if not constants:
        raise RuntimeError("challenge3 constant not found in CHECK_CH3 function.")
    counts: dict[int, int] = {}
    for value in constants:
        counts[value] = counts.get(value, 0) + 1
    ranked = sorted(counts.items(), key=lambda item: (-item[1], item[0]))
    return ranked[0][0]


def recover_runtime_layout(so_bytes: bytes) -> dict[str, int]:
    with ELFFileBytes(so_bytes) as elf:
        text = elf.get_section_by_name(".text")
        if text is None:
            raise SystemExit(".text not found in libzkpcore.so.")
        text_addr = text["sh_addr"]
        text_data = text.data()

    def word_at(addr: int) -> int:
        return read_u32_le(text_data, addr - text_addr)

    auth_page = decode_adrp(word_at(FN_NATIVE_INIT + OFF_INIT_ADRP_AUTH), FN_NATIVE_INIT + OFF_INIT_ADRP_AUTH)
    state_page = decode_adrp(word_at(FN_NATIVE_INIT + OFF_INIT_ADRP_STATE), FN_NATIVE_INIT + OFF_INIT_ADRP_STATE)
    ready_page = decode_adrp(word_at(FN_NATIVE_INIT + OFF_INIT_ADRP_READY), FN_NATIVE_INIT + OFF_INIT_ADRP_READY)
    token_page = decode_adrp(word_at(FN_CHECK_CH3_CORE + OFF_CH3_TOKEN_ADRP), FN_CHECK_CH3_CORE + OFF_CH3_TOKEN_ADRP)
    token_off = decode_add_imm(word_at(FN_CHECK_CH3_CORE + OFF_CH3_TOKEN_ADRP + 4))

    return {
        "init_done": ready_page + 0x18,
        "auth_ptr": auth_page + 0x20,
        "state_ptr": state_page + 0x28,
        "token_buf": token_page + token_off,
    }


class NativeEmulator:
    def __init__(self, so_bytes: bytes):
        self.mu = Uc(UC_ARCH_ARM64, UC_MODE_ARM)
        self.heap = HEAP_BASE
        self.trace: list[int] = []

        with ELFFileBytes(so_bytes) as elf:
            for seg in elf.iter_segments():
                if seg["p_type"] != "PT_LOAD":
                    continue
                va = seg["p_vaddr"]
                memsz = seg["p_memsz"]
                start = align_down(va)
                end = align_up(va + memsz)
                try:
                    self.mu.mem_map(start, end - start)
                except UcError:
                    pass
                self.mu.mem_write(va, seg.data()[: seg["p_filesz"]])

        self.mu.mem_map(HEAP_BASE, 0x400000)
        self.mu.mem_map(STACK_BASE, STACK_SIZE)
        self.mu.mem_map(align_down(RET_ADDR), 0x1000)

        self.hooks = {
            0x5AF0: self.h_log,
            0x5B00: self.h_stack_fail,
            0x5B10: self.h_calloc,
            0x5B20: self.h_memset,
            0x5B30: self.h_free,
            0x5B40: self.h_strcmp,
            0x5B50: self.h_strlen,
            0x5B60: self.h_memcpy,
            0x5B70: self.h_strcpy,
            0x5B80: self.h_strcat,
            0x5BA0: self.h_strtoul,
            0x5BB0: self.h_strncpy,
            0x5BC0: self.h_strncpy,
            0x5CE0: self.h_new_array,
            0x5CF0: self.h_free,
            0x2D88: self.h_vsnprintf_wrapper,
            0x3270: self.h_vsnprintf_wrapper,
            0x42D4: self.h_vsnprintf_wrapper,
            0x4E58: self.h_sha256,
            0x59CC: self.h_hex,
        }
        self.mu.hook_add(UC_HOOK_CODE, self.hook_code)

    def reg(self, register: int) -> int:
        return self.mu.reg_read(register)

    def setreg(self, register: int, value: int) -> None:
        self.mu.reg_write(register, value & ((1 << 64) - 1))

    def hook_code(self, mu, address, size, user_data):
        self.trace.append(address)
        self.trace = self.trace[-40:]
        if address == RET_ADDR:
            raise UcError(UC_ERR_OK)
        handler = self.hooks.get(address)
        if handler:
            handler()
            self.ret()

    def ret(self) -> None:
        self.setreg(UC_ARM64_REG_PC, self.reg(UC_ARM64_REG_LR))

    def malloc(self, size: int) -> int:
        ptr = align_up(self.heap, 0x10)
        self.heap = ptr + align_up(size, 0x10)
        return ptr

    def alloc_str(self, text: str | bytes) -> int:
        data = text.encode() if isinstance(text, str) else text
        ptr = self.malloc(len(data) + 1)
        self.mu.mem_write(ptr, data + b"\x00")
        return ptr

    def cstr(self, address: int, limit: int = 0x10000) -> bytes:
        out = bytearray()
        for i in range(limit):
            b = self.mu.mem_read(address + i, 1)[0]
            if b == 0:
                break
            out.append(b)
        return bytes(out)

    def cstr_text(self, address: int) -> str:
        return self.cstr(address).decode("utf-8", "replace")

    def write_cstr(self, address: int, data: str | bytes) -> None:
        blob = data.encode() if isinstance(data, str) else data
        self.mu.mem_write(address, blob + b"\x00")

    def call(self, address: int, *args: int) -> int:
        regs = [
            UC_ARM64_REG_X0,
            UC_ARM64_REG_X1,
            UC_ARM64_REG_X2,
            UC_ARM64_REG_X3,
            UC_ARM64_REG_X4,
            UC_ARM64_REG_X5,
            UC_ARM64_REG_X6,
            UC_ARM64_REG_X7,
        ]
        for reg, arg in zip(regs, args):
            self.setreg(reg, arg)
        self.setreg(UC_ARM64_REG_SP, STACK_BASE + STACK_SIZE // 2)
        self.setreg(UC_ARM64_REG_LR, RET_ADDR)
        try:
            self.mu.emu_start(address, RET_ADDR)
        except UcError as exc:
            if exc.errno != UC_ERR_OK:
                recent = ", ".join(hex(x) for x in self.trace[-12:])
                raise RuntimeError(
                    f"emulation failed at {hex(self.reg(UC_ARM64_REG_PC))}; last={recent}"
                ) from exc
        return self.reg(UC_ARM64_REG_X0)

    def h_log(self):
        self.setreg(UC_ARM64_REG_X0, 0)

    def h_stack_fail(self):
        raise RuntimeError("native stack check failed")

    def h_calloc(self):
        size = self.reg(UC_ARM64_REG_X0) * self.reg(UC_ARM64_REG_X1)
        ptr = self.malloc(size)
        self.mu.mem_write(ptr, b"\x00" * size)
        self.setreg(UC_ARM64_REG_X0, ptr)

    def h_new_array(self):
        ptr = self.malloc(self.reg(UC_ARM64_REG_X0))
        self.setreg(UC_ARM64_REG_X0, ptr)

    def h_free(self):
        self.setreg(UC_ARM64_REG_X0, 0)

    def h_memset(self):
        dst = self.reg(UC_ARM64_REG_X0)
        val = self.reg(UC_ARM64_REG_W1) & 0xFF
        size = self.reg(UC_ARM64_REG_X2)
        self.mu.mem_write(dst, bytes([val]) * size)
        self.setreg(UC_ARM64_REG_X0, dst)

    def h_memcpy(self):
        dst = self.reg(UC_ARM64_REG_X0)
        src = self.reg(UC_ARM64_REG_X1)
        size = self.reg(UC_ARM64_REG_X2)
        self.mu.mem_write(dst, bytes(self.mu.mem_read(src, size)))
        self.setreg(UC_ARM64_REG_X0, dst)

    def h_strlen(self):
        self.setreg(UC_ARM64_REG_X0, len(self.cstr(self.reg(UC_ARM64_REG_X0))))

    def h_strcmp(self):
        left = self.cstr(self.reg(UC_ARM64_REG_X0))
        right = self.cstr(self.reg(UC_ARM64_REG_X1))
        self.setreg(UC_ARM64_REG_X0, 0 if left == right else 1)

    def h_strcpy(self):
        dst = self.reg(UC_ARM64_REG_X0)
        self.write_cstr(dst, self.cstr(self.reg(UC_ARM64_REG_X1)))
        self.setreg(UC_ARM64_REG_X0, dst)

    def h_strcat(self):
        dst = self.reg(UC_ARM64_REG_X0)
        self.write_cstr(dst, self.cstr(dst) + self.cstr(self.reg(UC_ARM64_REG_X1)))
        self.setreg(UC_ARM64_REG_X0, dst)

    def h_strncpy(self):
        dst = self.reg(UC_ARM64_REG_X0)
        src = self.reg(UC_ARM64_REG_X1)
        size = self.reg(UC_ARM64_REG_X2)
        data = self.cstr(src, size)[:size]
        self.mu.mem_write(dst, data + b"\x00" * max(0, size - len(data)))
        self.setreg(UC_ARM64_REG_X0, dst)

    def h_strtoul(self):
        text = self.cstr_text(self.reg(UC_ARM64_REG_X0)).strip()
        base = self.reg(UC_ARM64_REG_X2) or 10
        try:
            value = int(text, base)
        except ValueError:
            value = 0
        self.setreg(UC_ARM64_REG_X0, value)

    def h_vsnprintf_wrapper(self):
        dst = self.reg(UC_ARM64_REG_X0)
        size = self.reg(UC_ARM64_REG_X1)
        fmt = self.cstr_text(self.reg(UC_ARM64_REG_X3))
        if fmt == "%s%s%s":
            out = (
                self.cstr(self.reg(UC_ARM64_REG_X4))
                + self.cstr(self.reg(UC_ARM64_REG_X5))
                + self.cstr(self.reg(UC_ARM64_REG_X6))
            )
        elif fmt == "%s/hint.txt":
            out = self.cstr(self.reg(UC_ARM64_REG_X4)) + b"/hint.txt"
        elif fmt == "ISCC{%s}":
            out = b"ISCC{" + self.cstr(self.reg(UC_ARM64_REG_X4)) + b"}"
        else:
            raise RuntimeError(f"unsupported native format string: {fmt!r}")
        self.mu.mem_write(dst, out[: max(0, size - 1)] + b"\x00")
        self.setreg(UC_ARM64_REG_X0, len(out))

    def h_sha256(self):
        src = self.reg(UC_ARM64_REG_X0)
        size = self.reg(UC_ARM64_REG_X1)
        dst = self.reg(UC_ARM64_REG_X2)
        self.mu.mem_write(dst, sha256(bytes(self.mu.mem_read(src, size))))
        self.setreg(UC_ARM64_REG_X0, dst)

    def h_hex(self):
        src = self.reg(UC_ARM64_REG_X0)
        size = self.reg(UC_ARM64_REG_X1)
        dst = self.reg(UC_ARM64_REG_X2)
        self.mu.mem_write(dst, bytes(self.mu.mem_read(src, size)).hex().encode() + b"\x00")
        self.setreg(UC_ARM64_REG_X0, dst)


def discover_challenge1_via_native(
    emu: NativeEmulator,
    rodata: bytes,
    seed: int,
    consts: tuple[int, int, int, int],
) -> str:
    def try_targets(targets: set[int]) -> str | None:
        for target in sorted(targets):
            cand = challenge1_hex_from_params(target, consts, lfsr_seed=seed)
            test_state = emu.call(FN_ALLOC_STATE)
            if emu.call(FN_CHECK_CH1_CORE, emu.alloc_str(cand), test_state) == 1:
                return cand
        return None

    aligned = {
        struct.unpack_from("<I", rodata, off)[0]
        for off in range(0, len(rodata) - 3, 4)
    }
    hit = try_targets(aligned)
    if hit is not None:
        return hit

    unaligned = {
        struct.unpack_from("<I", rodata, off)[0]
        for off in range(0, len(rodata) - 3)
    }
    hit = try_targets(unaligned - aligned)
    if hit is not None:
        return hit

    for off in range(0, len(rodata) - 20):
        consts = struct.unpack_from("<IIII", rodata, off)
        target = struct.unpack_from("<I", rodata, off + 16)[0]
        cand = challenge1_hex_from_params(target, consts, lfsr_seed=seed)
        test_state = emu.call(FN_ALLOC_STATE)
        if emu.call(FN_CHECK_CH1_CORE, emu.alloc_str(cand), test_state) == 1:
            return cand

    raise RuntimeError("challenge1 candidate not found in .rodata scan")


def recover_challenge3(token: str, xor_const: int) -> str:
    packed = 0
    for byte in token.encode()[:8]:
        packed = ((packed & ((1 << 56) - 1)) << 8) | byte
    mixed = packed ^ xor_const
    return sha256(mixed.to_bytes(8, "little")).hex()[:16]


def solve(apk_path: Path) -> dict[str, object]:
    so_bytes, so_name = read_so_from_apk(apk_path)
    rodata = extract_rodata(so_bytes)
    globals_info = recover_runtime_layout(so_bytes)
    xor_const = recover_challenge3_constant(so_bytes)
    ch1_params = recover_challenge1_params(so_bytes)

    username = recover_username(rodata)
    password = recover_password(rodata)

    emu = NativeEmulator(so_bytes)
    auth = emu.call(FN_ALLOC_AUTH)
    state = emu.call(FN_ALLOC_STATE)

    emu.mu.mem_write(globals_info["init_done"], b"\x01")
    emu.mu.mem_write(globals_info["auth_ptr"], struct.pack("<Q", auth))
    emu.mu.mem_write(globals_info["state_ptr"], struct.pack("<Q", state))

    login_ok = emu.call(FN_LOGIN_CORE, auth, emu.alloc_str(username), emu.alloc_str(password))
    if login_ok != 1:
        raise RuntimeError(f"native login failed for {username!r}/{password!r}")

    challenge1 = challenge1_hex_from_params(
        int(ch1_params["target"]),
        tuple(ch1_params["consts"]),
        lfsr_seed=int(ch1_params["seed"]),
    )
    if emu.call(FN_CHECK_CH1_CORE, emu.alloc_str(challenge1), emu.call(FN_ALLOC_STATE)) != 1:
        challenge1 = discover_challenge1_via_native(
            emu,
            rodata,
            int(ch1_params["seed"]),
            tuple(ch1_params["consts"]),
        )

    if emu.call(FN_CHECK_CH1_CORE, emu.alloc_str(challenge1), state) != 1:
        raise RuntimeError(f"challenge1 check failed: {challenge1}")
    emu.mu.mem_write(state, struct.pack("<I", 1) + b"\x01\x00\x00")

    ch2_buf = emu.malloc(64)
    emu.call(FN_MAKE_CH2, ch2_buf)
    challenge2 = emu.cstr_text(ch2_buf)
    if emu.call(FN_CHECK_CH2_CORE, emu.alloc_str(challenge2), state) != 1:
        raise RuntimeError(f"challenge2 check failed: {challenge2}")
    emu.mu.mem_write(state, struct.pack("<I", 2) + b"\x01\x01\x00")

    token = emu.cstr_text(globals_info["token_buf"])
    challenge3 = recover_challenge3(token, xor_const)
    if emu.call(FN_CHECK_CH3_CORE, emu.alloc_str(challenge3), emu.alloc_str(token), state) != 1:
        raise RuntimeError(f"challenge3 check failed: {challenge3}")
    emu.mu.mem_write(state, struct.pack("<I", 3) + b"\x01\x01\x01")

    flag_buf = emu.malloc(256)
    gen_ok = emu.call(FN_GEN_FLAG_CORE, state, auth, flag_buf, 0x80)
    if gen_ok != 1:
        raise RuntimeError("native flag generation failed")

    return {
        "apk": str(apk_path),
        "so": so_name,
        "username": username,
        "password": password,
        "challenge1": challenge1,
        "challenge2": challenge2,
        "intermediate_token": token,
        "challenge3_xor_const": f"0x{xor_const:016x}",
        "challenge3": challenge3,
        "flag": emu.cstr_text(flag_buf),
    }


def main() -> None:
    parser = argparse.ArgumentParser(description="Generic solver for the ISCC MOBILE challenge template used by attachment-25.apk.")
    parser.add_argument("apk", nargs="?", type=Path, help="APK path, defaults to first *.apk in cwd")
    parser.add_argument("--json", action="store_true", help="print JSON instead of key: value lines")
    parser.add_argument("--write-flag", action="store_true", help="write recovered flag to flag.txt")
    args = parser.parse_args()

    apk_path = args.apk or first_apk()
    result = solve(apk_path)
    if args.json:
        print(json.dumps(result, ensure_ascii=False, indent=2))
    else:
        for key in (
            "apk",
            "so",
            "username",
            "password",
            "challenge1",
            "challenge2",
            "intermediate_token",
            "challenge3_xor_const",
            "challenge3",
            "flag",
        ):
            print(f"{key}: {result[key]}")
    if args.write_flag:
        Path("flag.txt").write_text(str(result["flag"]) + "\n", encoding="utf-8")


if __name__ == "__main__":
    try:
        main()
    except Exception as exc:
        print(f"error: {exc}", file=sys.stderr)
        raise SystemExit(1) from exc
