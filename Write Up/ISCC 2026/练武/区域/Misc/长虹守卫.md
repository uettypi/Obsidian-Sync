
## 解题步骤

### Step 1: 从文本日志中提取线索 (Hint)
分析 `LX517.txt`，可以在文件中发现一串长度大于 80 的 Base64 编码字符串。
将其解码后，会发现一段包含关键信息的提示文本，其中包含两个核心字段：
*   **`ALGO:`**：暗示了后续使用的加密算法结构。
*   **`KEYMAT:`**：提供了生成密钥的模板。模板中包含三个占位符：`<EVT_A_UTC>`、`<EVT_B_UTC>` 和 `<EVT_C_UTC>`。

这说明我们需要找到事件 A、B、C 发生的 **UTC时间戳（Unix Epoch）**，填入模板中才能构造出正确的密钥材料 (Key Material)。

### Step 2: 从流量包中提取密文
分析 `LX517.pcap` 流量包。追踪 TCP 流，我们可以发现在 TCP 的 Payload 数据中，包含一段以 `CIPH:` 开头的字符串。
提取 `CIPH:` 后面的十六进制字符串并转换为字节流，这就是我们需要解密的**目标密文 (Ciphertext)**。

### Step 3: 分析日志寻找关键事件 (Event)
读取 `LX517.txt` 中的数据行，发现其为类似 CSV 格式的日志，格式大致为：
`序号, 本地时间(YYYY-MM-DD HH:MM:SS), 纬度(lat), 经度(lon), ..., 事件(event), 详情(detail)`

根据 Step 1 的线索，我们需要在日志中筛选出以下三个关键事件的时间、经纬度信息：
1.  **事件A**：`event` 字段为 `EVT_A` 的行。
2.  **事件B**：`event` 字段为 `EVT_B` 的行。
3.  **事件C**：`event` 字段为 `EVT_C`，且 `detail` 字段中包含 `marker=A2` 或 `marker=X9` 的行。

### Step 4: 地理位置与时区转换 (核心考点)
日志中记录的时间是**本地时间 (Local Time)**，而密钥模板需要的是 **UTC 时间戳**。因为设备（比如船舶）在不断移动，其所处的时区也在不断变化。

根据经纬度计算时区的规则：
1.  **基础时区计算**：将经度除以 15 并四舍五入，即 `round(lon / 15)`。
2.  **国际日期变更线 (IDL) 坑点**：日志记录的轨迹跨越了国际日期变更线（经度大于 150 或小于 -150 的区域）。在此区域内：
    *   经度 > 150 时，可能的时区有 +12, +13, +14。
    *   经度 < -150 时，可能的时区有 -10, -11, -12。
    *   由于跨越变更线可能存在日期计算的偏差，本地时间可能需要修正（0 天或减去 1 天）。

因此，针对筛选出的每一个事件行，我们需要进行一个小范围的穷举，计算出所有可能的 UTC Unix 时间戳作为候选值：
`UTC时间 = 本地时间 + 日期修正(0或-1天) - 经度推算的可能时区偏移(小时)`

### Step 5: 逆向自定义加密算法
根据 EXP 中的 `decrypt` 函数，可以看出题目使用的是基于 **SHA-256 的自定义计数器模式 (CTR) 流密码**：
1.  将构造好的完整 `KEYMAT` 字符串编码为字节。
2.  将一个 32 位的大端序整数 `i`（从 0 开始的块计数器）拼接到 `KEYMAT` 后面。
3.  计算拼接数据的 SHA-256 哈希值，作为该数据块的密钥流 (Keystream)。
4.  将生成的密钥流与密文进行异或 (`XOR`) 运算，得出明文。

### Step 6: 穷举爆破获取 Flag
将以上步骤结合起来，编写脚本进行爆破：

1.  遍历所有可能的 `EVT_A`、`EVT_B`、`EVT_C` 组合。
2.  对每种组合，遍历所有可能的时区和日期修正计算出的候选 UTC 时间戳。
3.  将三个时间戳替换入 `KEYMAT` 模板（`<EVT_A_UTC>` 等）。
4.  使用生成的 `KEYMAT` 进行 SHA-256 流密码解密。
5.  使用正则表达式 `ISCC\{[0-9a-fA-F]{32}\}` 检查解密后的明文。如果匹配成功，则爆破结束，输出 Flag。

```python
import base64
import hashlib
import re
import struct
from datetime import datetime, timedelta, timezone
from pathlib import Path


ROOT = Path(__file__).resolve().parent
TXT = ROOT / "LX517.txt"
PCAP = ROOT / "LX517.pcap"


def decode_hint(text: str) -> str:
    for token in re.findall(r"[A-Za-z0-9+/=]{80,}", text):
        try:
            decoded = base64.b64decode(token).decode("utf-8")
        except Exception:
            continue
        if "KEYMAT:" in decoded and "ALGO:" in decoded:
            return decoded
    raise RuntimeError("Base64 hint not found")


def parse_rows(text: str):
    rows = []
    for line in text.splitlines():
        if not re.match(r"^\d{4},", line):
            continue
        parts = line.split(",", 9)
        if len(parts) != 10:
            continue
        rows.append(
            {
                "idx": int(parts[0]),
                "local_dt": datetime.strptime(parts[1], "%Y-%m-%d %H:%M:%S"),
                "lat": float(parts[2]),
                "lon": float(parts[3]),
                "event": parts[8],
                "detail": parts[9],
            }
        )
    return rows


def likely_tz_offsets(lon: float):
    """Return plausible UTC offsets for this challenge's IDL route."""
    offsets = {round(lon / 15)}
    if lon > 150:
        offsets.update({12, 13, 14})
    if lon < -150:
        offsets.update({-10, -11, -12})
    return sorted(offsets)


def epoch_candidates(row):
    candidates = set()
    for utc_offset_hours in likely_tz_offsets(row["lon"]):
        # The note mentions a date correction around the IDL.  The released
        # attachment is solved by trying both interpretations.
        for date_corr_days in (0, -1):
            local_dt = row["local_dt"] + timedelta(days=date_corr_days)
            utc_dt = local_dt - timedelta(hours=utc_offset_hours)
            candidates.add(int(utc_dt.replace(tzinfo=timezone.utc).timestamp()))
    return sorted(candidates)


def tcp_payloads_from_pcap(path: Path):
    data = path.read_bytes()
    if len(data) < 24:
        return []

    magic = data[:4]
    if magic == b"\xd4\xc3\xb2\xa1":
        endian = "<"
    elif magic == b"\xa1\xb2\xc3\xd4":
        endian = ">"
    else:
        raise RuntimeError("Unsupported pcap magic")

    payloads = []
    pos = 24
    while pos + 16 <= len(data):
        _ts_sec, _ts_usec, incl_len, _orig_len = struct.unpack(
            endian + "IIII", data[pos : pos + 16]
        )
        pos += 16
        pkt = data[pos : pos + incl_len]
        pos += incl_len

        if len(pkt) < 14:
            continue
        eth_type = struct.unpack("!H", pkt[12:14])[0]
        if eth_type != 0x0800:
            continue

        ip = pkt[14:]
        if len(ip) < 20 or ip[0] >> 4 != 4:
            continue
        ihl = (ip[0] & 0x0F) * 4
        total_len = struct.unpack("!H", ip[2:4])[0]
        if ip[9] != 6 or len(ip) < total_len:
            continue

        tcp = ip[ihl:total_len]
        if len(tcp) < 20:
            continue
        data_offset = (tcp[12] >> 4) * 4
        payload = tcp[data_offset:]
        if payload:
            payloads.append(payload)
    return payloads


def extract_ciphertext(path: Path) -> bytes:
    joined = b"".join(tcp_payloads_from_pcap(path))
    match = re.search(rb"CIPH:([0-9a-fA-F]+)", joined)
    if not match:
        match = re.search(rb"CIPH:([0-9a-fA-F]+)", path.read_bytes())
    if not match:
        raise RuntimeError("CIPH payload not found")
    return bytes.fromhex(match.group(1).decode())


def decrypt(ciphertext: bytes, keymat: str) -> bytes:
    blocks = (len(ciphertext) + 31) // 32
    keystream = b"".join(
        hashlib.sha256(keymat.encode() + struct.pack(">I", i)).digest()
        for i in range(blocks)
    )
    return bytes(c ^ k for c, k in zip(ciphertext, keystream))


def main():
    text = TXT.read_text(encoding="utf-8", errors="ignore")
    hint = decode_hint(text)
    rows = parse_rows(text)
    ciphertext = extract_ciphertext(PCAP)

    key_template = re.search(r"^KEYMAT:(.+)$", hint, re.MULTILINE).group(1).strip()
    evt_a_rows = [r for r in rows if r["event"] == "EVT_A"]
    evt_b_rows = [r for r in rows if r["event"] == "EVT_B"]
    evt_c_rows = [
        r
        for r in rows
        if r["event"] == "EVT_C" and ("marker=A2" in r["detail"] or "marker=X9" in r["detail"])
    ]

    tried = 0
    for evt_a in evt_a_rows:
        for evt_b in evt_b_rows:
            for evt_c in evt_c_rows:
                for syn in epoch_candidates(evt_a):
                    for idl in epoch_candidates(evt_b):
                        for a2 in epoch_candidates(evt_c):
                            keymat = (
                                key_template.replace("<EVT_A_UTC>", str(syn))
                                .replace("<EVT_B_UTC>", str(idl))
                                .replace("<EVT_C_UTC>", str(a2))
                            )
                            tried += 1
                            plain = decrypt(ciphertext, keymat)
                            if re.fullmatch(rb"ISCC\{[0-9a-fA-F]{32}\}", plain):
                                print("hint:")
                                print(hint)
                                print("ciphertext:", ciphertext.hex())
                                print("keymat:", keymat)
                                print("flag:", plain.decode())
                                print("tried:", tried)
                                return

    raise RuntimeError(f"Flag not found, tried {tried} candidates")


if __name__ == "__main__":
    main()

```
