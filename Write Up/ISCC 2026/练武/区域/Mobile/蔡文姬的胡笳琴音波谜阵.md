
### 1. APK 静态分析与资源提取
首先需要对目标 APK（`app-release.apk`）进行解包，提取关键的验证要素和资源：
*   **多媒体资源长度**：在 `res/` 目录下存在一个 `.mp4` 文件（可能与蔡文姬动画相关），Java 层通过读取这个文件的实际字节长度（`resource_len`）来参与后续的 Hash 运算。
*   **缓存数据库**：提取 assets 目录下的加密数据库文件 `assets/appview_cache.db`。
*   **Native动态库**：提取 `lib/x86_64/libmobile04.so`。
*   **DEX 常量分析**：
    *   在 `HarpMelodyManager`（胡笳琴音管理器）类中，提取按键对应的 Token 字符串数组（通常是8个，例如：`CWJ`, `XMAS`, `FLOW`, `STAR`, `ROSE`, `GOAL`, `DANCE`, `MAGI`）。
    *   在 `K1.g` 方法中提取一个用于计算的魔数（默认可能为 `0xDEADBEEF`）。
    *   提取 App 的包名（例如 `com.example.mobile04`）。

### 2. Java层逻辑分析：生成 CW 校验值
玩家在界面上点击琴键，会产生一个由 Token 拼接而成的琴音序列。系统会根据此序列生成一个关键的验证值 `cw`。生成过程涉及 Java 与 Native 的交互：
*   **计算种子 (Seed)**：
    计算公式：`derive_seed = java_hash_u32(package) ^ resource_len ^ K1_magic`。即将包名哈希、MP4文件大小、K1方法中的魔数进行异或。
*   **TEA 密钥派生**：
    Native 层库中导出了一个 `_n_derive_impl` 函数，其中硬编码了 4 个 32 位的常量（`derive_consts`）。这些常量与上面的 `derive_seed` 异或，生成了 4 个 32位整数，作为后续 TEA 加密的密钥（`tea_key`）。
*   **计算 CW 字符串**：
    1. 获取玩家弹奏的拼接字符串（长度在25-64之间）。
    2. 对字符串字节进行按位交换（`bit_swap_bytes`: `((b & 0x55) << 1) | ((b & 0xAA) >> 1)`）。
    3. 利用 FNV-1a 算法（`fnv1a64`）计算得到一个 64 位的哈希值。
    4. 将 64 位哈希分为两个 32 位整数 `v0` 和 `v1`，使用上述派生的 `tea_key` 进行 32 轮 TEA 加密。
    5. 将加密后的结果打包为十六进制，并附带 CRC32 校验，最终格式化为：`cw:<hex>:<crc32>`。

### 3. Native 层逆向：提取关键密钥与密文
逆向分析 `libmobile04.so`，可以从中提取出最终校验所需的多个关键元素：
*   **符号表读取**：
    *   `_K1`：一个字符串，用作简单的循环异或（Repeat-XOR）密钥。
    *   `_K2`：一个 16 字节的字符串，用作 RC6 加密的初始密钥。
    *   `_TGT`：使用 Base58 编码的最终目标密文。
*   **识别魔改 RC6 算法**：
    程序的 RC6 加密算法被修改了初始化常量。在类似 `_ZN3_C1C2Ejj` 的 C++ 构造函数中，对标准 RC6 参数 $P$ 和 $Q$ 进行了魔改。需要通过汇编特征（如 `\x81\xf1...`、`\x41\x81\xc6...` 等）提取出特定的异或值（`p_xor`）、常数（`q_const`）以及操作顺序（如先或后加 `or_then_add` 或先加后或 `add_then_or`）。

### 4. 揭秘 `appview_cache.db`
`assets/appview_cache.db` 并非标准的数据库，而是使用前文骤生成的 `cw` 值进行简单加密的缓存文件。
*   **解密算法**：取 `cw` 字符串与缓存文件的前 16 字节头部拼接，计算 CRC32。随后利用此 CRC32 与字节的偏移量规律进行循环异或解密。
*   **提取关键种子**：解密后的缓存文件实质是一块数据结构。我们需要从中提取出位于特定偏移位置的关键种子：偏移 `0x80` 处的 `seed1` 和 偏移 `0xB4` 处的 `seed2`。这两个种子是解开最终密文的关键。

### 5. RC6 解密与密文对比获取 Flag
这是验证环节的最后一步。
*   **目标密文预处理**：将 Native 层的 `_TGT` 进行 Base58 解码，如果不足32字节则补零，再使用 `_K1` 对其进行循环异或解密，得到两块（共32字节）的预期RC6密文（`expected_cipher`）。
*   **RC6 密钥扩展 (Schedule)**：将解密缓存得出的 `seed1`、`seed2` 混合输入，再结合 Native 提取出的 `_K2` 和**魔改参数**，生成用于 RC6 的 44 个 32位字（`s_words`）。
*   **解密验证**：利用生成的 `s_words`，对 32 字节进行 RC6 分块解密，解密出明文。

### 6. 解题思路：基于排列组合的暴力破解
因为真实的琴音序列就是那 8 个 Token 的某一种排列组合，由于排列组合数量有限（$8! = 40320$ 种），因此 EXP 使用穷举（Brute-force）的思路完成自动求解：
1. 全排列枚举所有的 Token 序列（如 `CWJXMASFLOWSTARROSEGOALDANCEMAGI` 等组合）。
2. 根据序列计算 `cw` 值。
3. 利用 `cw` 解密 `appview_cache.db` 获取 `seed1` 和 `seed2`。
4. 使用 `seed1` 和 `seed2` 衍生 RC6 密钥，解密并校验明文的“可打印字符得分”（`printable_score`）。
5. 如果解密出的内容都是可见的 ASCII 字符且具备意义（匹配分数阈值），且将其重新 RC6 加密后能完美匹配预期的 `expected_cipher`，则认为爆破成功。
6. 将解密出来的 ASCII 字符串与 Prefix（如 `ISCC{`） 拼接闭合，即可拿到最终的 Flag。

#!/usr/bin/env python3
"""
Solver for the mobile04-style ISCC Android reverse challenge.

It extracts constants from libmobile04.so, derives the Java-side cw value from
harp click sequences, decrypts assets/appview_cache.db, and reverses the native
RC6 check to recover the flag body.

Usage:
  python solve_mobile04_variant.py path/to/app-release.apk
  python solve_mobile04_variant.py path/to/app.apk --tokens CWJ,XMAS,FLOW,STAR,ROSE,GOAL,DANCE,MAGI
  python solve_mobile04_variant.py path/to/app.apk --seq CWJXMASFLOWSTARROSEGOALDANCEMAGI
"""

from __future__ import annotations

import argparse
import itertools
import os
import re
import struct
import sys
import tempfile
import zipfile
import zlib
from dataclasses import dataclass


MASK32 = 0xFFFFFFFF
DEFAULT_TOKENS = ("CWJ", "XMAS", "FLOW", "STAR", "ROSE", "GOAL", "DANCE", "MAGI")
BASE58_ALPHABET = "123456789ABCDEFGHJKLMNPQRSTUVWXYZabcdefghijkmnopqrstuvwxyz"
DEFAULT_PACKAGE = "com.example.mobile04"
DEFAULT_PREFIX = "ISCC{"
DEFAULT_K1_MAGIC = 0xDEADBEEF


@dataclass(frozen=True)
class Rc6Params:
    p_xor: int = 0x4A205268
    q_const: int = (-0x45DB0748) & MASK32
    q_mode: str = "or_then_add"


def u32(x: int) -> int:
    return x & MASK32


def rol(x: int, n: int) -> int:
    n &= 31
    x &= MASK32
    if n == 0:
        return x
    return ((x << n) | (x >> (32 - n))) & MASK32


def ror(x: int, n: int) -> int:
    n &= 31
    x &= MASK32
    if n == 0:
        return x
    return ((x >> n) | (x << (32 - n))) & MASK32


def java_hash_u32(s: str) -> int:
    h = 0
    for ch in s:
        h = u32(31 * h + ord(ch))
    return h


def bit_swap_bytes(data: bytes) -> bytes:
    return bytes((((b & 0x55) << 1) | ((b & 0xAA) >> 1)) & 0xFF for b in data)


def fnv1a64(data: bytes) -> int:
    h = (-3750763034362895579) & ((1 << 64) - 1)
    for b in data:
        h ^= b
        h = (h * 1099511628211) & ((1 << 64) - 1)
    return h


def read_uleb128(data: bytes, off: int) -> tuple[int, int]:
    result = 0
    shift = 0
    while True:
        b = data[off]
        off += 1
        result |= (b & 0x7F) << shift
        if b < 0x80:
            return result, off
        shift += 7


def tea_encrypt(v0: int, v1: int, key: tuple[int, int, int, int]) -> tuple[int, int]:
    s = 0
    delta = u32(-1640531527)
    for _ in range(32):
        s = u32(s + delta)
        v0 = u32(v0 + (((v1 << 4) + key[0]) ^ (v1 + s) ^ ((v1 >> 5) + key[1])))
        v1 = u32(v1 + (((v0 << 4) + key[2]) ^ (v0 + s) ^ ((v0 >> 5) + key[3])))
    return v0, v1


def cw_from_sequence(seq: str, tea_key: tuple[int, int, int, int]) -> str:
    if not (25 <= len(seq) <= 64):
        return "cw:err_len:00000000"
    h = fnv1a64(bit_swap_bytes(seq.encode()))
    v0, v1 = h & MASK32, (h >> 32) & MASK32
    e0, e1 = tea_encrypt(v0, v1, tea_key)
    out = struct.pack("<II", e0, e1)
    return f"cw:{out.hex()}:{zlib.crc32(out) & MASK32:08x}"


def base58_decode(s: str) -> bytes:
    n = 0
    for ch in s:
        n = n * 58 + BASE58_ALPHABET.index(ch)
    out = n.to_bytes((n.bit_length() + 7) // 8, "big") if n else b""
    pad = len(s) - len(s.lstrip("1"))
    return b"\x00" * pad + out


def normalize_target_bytes(target: str) -> bytes:
    decoded = base58_decode(target)
    if len(decoded) < 32:
        decoded = b"\x00" * (32 - len(decoded)) + decoded
    elif len(decoded) > 32:
        decoded = decoded[-32:]
    return decoded


def decrypt_cache(cache: bytes, cw: str) -> tuple[bytes, int]:
    header = cache[:16]
    enc = bytearray(cache[16:])
    crc = zlib.crc32(cw.encode() + header) & MASK32
    for i in range(len(enc)):
        enc[i] ^= (((crc >> ((i & 3) * 8)) & 0xFF) + i) & 0xFF
    return bytes(enc), crc


def rc6_schedule(seed1: int, seed2: int, key2: bytes, params: Rc6Params | None = None) -> list[int]:
    if len(key2) != 16:
        raise ValueError(f"K2 must be 16 bytes, got {len(key2)}")
    params = params or Rc6Params()
    l_words = list(struct.unpack("<4I", key2))
    p = u32(rol(seed2, 13) ^ seed1 ^ params.p_xor)
    q_base = u32(rol(seed2, 25) ^ seed1)
    if params.q_mode == "add_then_or":
        q = u32(q_base + params.q_const) | 1
    elif params.q_mode == "or_then_add":
        q = u32((q_base | 1) + params.q_const)
    else:
        raise ValueError(f"unsupported RC6 q mode: {params.q_mode}")
    s_words = [u32(p + i * q) for i in range(44)]
    a = b = i = j = 0
    for _ in range(132):
        a = s_words[i] = rol(u32(s_words[i] + a + b), 3)
        b = l_words[j] = rol(u32(l_words[j] + a + b), u32(a + b))
        i = (i + 1) % 44
        j = (j + 1) % 4
    return s_words


def rc6_decrypt_block(block: bytes, s_words: list[int]) -> bytes:
    a, b, c, d = struct.unpack("<4I", block)
    c = u32(c - s_words[43])
    a = u32(a - s_words[42])
    for i in range(20, 0, -1):
        a, b, c, d = d, a, b, c
        u = rol(u32(d * u32(2 * d + 1)), 5)
        t = rol(u32(b * u32(2 * b + 1)), 5)
        c = u32(ror(u32(c - s_words[2 * i + 1]), t) ^ u)
        a = u32(ror(u32(a - s_words[2 * i]), u) ^ t)
    d = u32(d - s_words[1])
    b = u32(b - s_words[0])
    return struct.pack("<4I", a, b, c, d)


def rc6_encrypt_block(block: bytes, s_words: list[int]) -> bytes:
    a, b, c, d = struct.unpack("<4I", block)
    b = u32(b + s_words[0])
    d = u32(d + s_words[1])
    for i in range(1, 21):
        t = rol(u32(b * u32(2 * b + 1)), 5)
        u = rol(u32(d * u32(2 * d + 1)), 5)
        a = u32(rol(a ^ t, u) + s_words[2 * i])
        c = u32(rol(c ^ u, t) + s_words[2 * i + 1])
        a, b, c, d = b, c, d, a
    a = u32(a + s_words[42])
    c = u32(c + s_words[43])
    return struct.pack("<4I", a, b, c, d)


def repeat_xor(data: bytes, key: bytes) -> bytes:
    return bytes(b ^ key[i % len(key)] for i, b in enumerate(data))


def printable_score(data: bytes) -> int:
    if any(b < 0x20 or b > 0x7E for b in data):
        return -1000
    text = data.decode("ascii", errors="ignore")
    good = sum(ch.isalnum() or ch in "_-?%&+*=@!#." for ch in text)
    bad = len(text) - good
    score = 100 + good * 3 - bad * 8
    if " " not in text:
        score += 20
    if re.search(r"[A-Za-z]{4,}", text):
        score += 15
    return score


@dataclass
class Section:
    name: str
    addr: int
    offset: int
    size: int
    entsize: int


class Elf64LE:
    def __init__(self, data: bytes):
        self.data = data
        if data[:4] != b"\x7FELF" or data[4] != 2 or data[5] != 1:
            raise ValueError("only little-endian ELF64 is supported")
        e_shoff = struct.unpack_from("<Q", data, 0x28)[0]
        e_shentsize = struct.unpack_from("<H", data, 0x3A)[0]
        e_shnum = struct.unpack_from("<H", data, 0x3C)[0]
        e_shstrndx = struct.unpack_from("<H", data, 0x3E)[0]
        raw_sections = []
        for i in range(e_shnum):
            off = e_shoff + i * e_shentsize
            raw_sections.append(struct.unpack_from("<IIQQQQIIQQ", data, off))
        shstr = raw_sections[e_shstrndx]
        shstr_data = data[shstr[4] : shstr[4] + shstr[5]]
        self.sections: list[Section] = []
        for sh in raw_sections:
            name_off = sh[0]
            end = shstr_data.find(b"\x00", name_off)
            name = shstr_data[name_off:end].decode("ascii", errors="replace") if end >= 0 else ""
            self.sections.append(Section(name=name, addr=sh[3], offset=sh[4], size=sh[5], entsize=sh[9]))
        self.symbols = self._parse_dynsym()
        self.relocations = self._parse_relocations()

    def section(self, name: str) -> Section | None:
        return next((s for s in self.sections if s.name == name), None)

    def va_to_offset(self, va: int) -> int:
        for s in self.sections:
            if s.size and s.addr <= va < s.addr + s.size:
                return s.offset + (va - s.addr)
        raise ValueError(f"VA 0x{va:x} is not backed by a file section")

    def read_va(self, va: int, size: int) -> bytes:
        off = self.va_to_offset(va)
        return self.data[off : off + size]

    def read_cstr_va(self, va: int, max_len: int = 256) -> str:
        off = self.va_to_offset(va)
        end = self.data.find(b"\x00", off, min(len(self.data), off + max_len))
        if end < 0:
            end = min(len(self.data), off + max_len)
        return self.data[off:end].decode("ascii", errors="replace")

    def _parse_dynsym(self) -> dict[str, tuple[int, int]]:
        dynsym = self.section(".dynsym")
        dynstr = self.section(".dynstr")
        if not dynsym or not dynstr or not dynsym.entsize:
            return {}
        strings = self.data[dynstr.offset : dynstr.offset + dynstr.size]
        out = {}
        for off in range(dynsym.offset, dynsym.offset + dynsym.size, dynsym.entsize):
            st_name, _info, _other, _shndx, st_value, st_size = struct.unpack_from("<IBBHQQ", self.data, off)
            if st_name == 0:
                continue
            end = strings.find(b"\x00", st_name)
            if end < 0:
                continue
            name = strings[st_name:end].decode("utf-8", errors="replace")
            out[name] = (st_value, st_size)
        return out

    def _parse_relocations(self) -> dict[int, int]:
        out = {}
        for sec_name in (".rela.dyn", ".rela.plt"):
            sec = self.section(sec_name)
            if not sec or not sec.entsize:
                continue
            for off in range(sec.offset, sec.offset + sec.size, sec.entsize):
                r_offset, r_info, r_addend = struct.unpack_from("<QQq", self.data, off)
                r_type = r_info & 0xFFFFFFFF
                if r_type == 8:  # R_X86_64_RELATIVE
                    out[r_offset] = r_addend
        return out

    def symbol_addr(self, needle: str) -> tuple[int, int]:
        for name, value in self.symbols.items():
            if name == needle:
                return value
        for name, value in self.symbols.items():
            if name.startswith(needle):
                return value
        # Function symbols are C++-mangled in this challenge family
        # (for example _Z14_n_derive_implP7_JNIEnvP7_jclassi).
        relaxed = needle.lstrip("_")
        for name, value in self.symbols.items():
            if relaxed in name:
                return value
        raise KeyError(f"missing symbol {needle}")

    def pointer_symbol_string(self, sym_name: str) -> str:
        addr, _size = self.symbol_addr(sym_name)
        if addr in self.relocations:
            ptr = self.relocations[addr]
        else:
            ptr = struct.unpack("<Q", self.read_va(addr, 8))[0]
        return self.read_cstr_va(ptr)

    def all_ascii_strings(self, min_len: int = 4) -> list[tuple[int, str]]:
        out = []
        for m in re.finditer(rb"[\x20-\x7e]{%d,}" % min_len, self.data):
            try:
                va = None
                off = m.start()
                for s in self.sections:
                    if s.offset <= off < s.offset + s.size:
                        va = s.addr + (off - s.offset)
                        break
                out.append((va or off, m.group().decode("ascii", errors="replace")))
            except UnicodeDecodeError:
                pass
        return out


def extract_rc6_params(elf: Elf64LE) -> Rc6Params:
    try:
        ctor_addr, ctor_size = elf.symbol_addr("_ZN3_C1C2Ejj")
        ctor = elf.read_va(ctor_addr, min(ctor_size or 0x300, 0x400))
    except Exception:
        return Rc6Params()

    p_match = re.search(rb"\x81\xf1(.{4})", ctor, re.S)
    p_xor = struct.unpack("<I", p_match.group(1))[0] if p_match else Rc6Params().p_xor

    add_then_or = re.search(rb"\x41\x81\xc6(.{4})\x41\x83\xce\x01", ctor, re.S)
    if add_then_or:
        return Rc6Params(p_xor=p_xor, q_const=struct.unpack("<I", add_then_or.group(1))[0], q_mode="add_then_or")

    or_then_add = re.search(rb"\x41\x83\xce\x01\x41\x8d\x96(.{4})", ctor, re.S)
    if or_then_add:
        return Rc6Params(p_xor=p_xor, q_const=struct.unpack("<I", or_then_add.group(1))[0], q_mode="or_then_add")

    return Rc6Params(p_xor=p_xor)


def extract_native_constants(so_data: bytes) -> tuple[bytes, bytes, str, tuple[int, int, int, int], str | None, Rc6Params]:
    elf = Elf64LE(so_data)
    k1 = elf.pointer_symbol_string("_K1").encode()
    k2 = elf.pointer_symbol_string("_K2").encode()
    target = elf.pointer_symbol_string("_TGT")
    rc6_params = extract_rc6_params(elf)

    func_addr, func_size = elf.symbol_addr("_n_derive_impl")
    scan_size = func_size or 0x100
    func = elf.read_va(func_addr, min(scan_size, 0x200))
    consts = None
    marker = b"\x66\x0f\xef\x05"
    idx = func.find(marker)
    if idx >= 0:
        disp = struct.unpack_from("<i", func, idx + 4)[0]
        const_va = func_addr + idx + 8 + disp
        consts = struct.unpack("<4I", elf.read_va(const_va, 16))
    if consts is None:
        raise ValueError("failed to extract Z1.d xor constants from _n_derive_impl")

    package = None
    for _va, s in elf.all_ascii_strings():
        m = re.search(r"com/[A-Za-z0-9_/]+/Z1", s)
        if m:
            package = m.group(0).replace("/", ".").removesuffix(".Z1")
            break
    return k1, k2, target, tuple(consts), package, rc6_params


def choose_member(names: list[str], patterns: list[str]) -> str | None:
    lowered = {n: n.lower() for n in names}
    for pat in patterns:
        rx = re.compile(pat, re.I)
        matches = [n for n in names if rx.search(lowered[n])]
        if matches:
            return sorted(matches, key=lambda x: (len(x), x))[0]
    return None


def dex_strings(data: bytes) -> list[str]:
    if not data.startswith(b"dex\n"):
        return []
    string_ids_size, string_ids_off = struct.unpack_from("<II", data, 0x38)
    strings = []
    for i in range(string_ids_size):
        off = struct.unpack_from("<I", data, string_ids_off + i * 4)[0]
        _utf16_len, pos = read_uleb128(data, off)
        end = data.find(b"\x00", pos)
        if end == -1:
            strings.append("")
        else:
            strings.append(data[pos:end].decode("utf-8", errors="replace"))
    return strings


def extract_harp_tokens_from_dex(data: bytes) -> tuple[str, ...] | None:
    try:
        strings = dex_strings(data)
        if not strings:
            return None
        type_ids_size, type_ids_off = struct.unpack_from("<II", data, 0x40)
        method_ids_size, method_ids_off = struct.unpack_from("<II", data, 0x58)
        class_defs_size, class_defs_off = struct.unpack_from("<II", data, 0x60)
    except struct.error:
        return None

    try:
        type_string_ids = [struct.unpack_from("<I", data, type_ids_off + i * 4)[0] for i in range(type_ids_size)]
        methods = [struct.unpack_from("<HHI", data, method_ids_off + i * 8) for i in range(method_ids_size)]
    except struct.error:
        return None

    def collect_methods(off: int, count: int) -> tuple[list[tuple[int, int, str]], int]:
        method_idx = 0
        out = []
        for _ in range(count):
            diff, off = read_uleb128(data, off)
            _access_flags, off = read_uleb128(data, off)
            code_off, off = read_uleb128(data, off)
            method_idx += diff
            if method_idx < len(methods):
                name_idx = methods[method_idx][2]
                method_name = strings[name_idx] if name_idx < len(strings) else ""
                out.append((method_idx, code_off, method_name))
        return out, off

    try:
        for i in range(class_defs_size):
            base = class_defs_off + i * 32
            class_idx, _access, _super, _interfaces, _source, _annotations, class_data_off, _static_values = struct.unpack_from(
                "<IIIIIIII", data, base
            )
            if class_idx >= len(type_string_ids):
                continue
            desc_idx = type_string_ids[class_idx]
            desc = strings[desc_idx] if desc_idx < len(strings) else ""
            if not desc.endswith("/HarpMelodyManager;") or not class_data_off:
                continue

            off = class_data_off
            static_fields_size, off = read_uleb128(data, off)
            instance_fields_size, off = read_uleb128(data, off)
            direct_methods_size, off = read_uleb128(data, off)
            virtual_methods_size, off = read_uleb128(data, off)

            for _ in range(static_fields_size + instance_fields_size):
                _field_idx_diff, off = read_uleb128(data, off)
                _access_flags, off = read_uleb128(data, off)

            direct_methods, off = collect_methods(off, direct_methods_size)
            virtual_methods, _off = collect_methods(off, virtual_methods_size)

            refs = []
            for _method_idx, code_off, _method_name in direct_methods + virtual_methods:
                if not code_off:
                    continue
                _registers, _ins, _outs, _tries, _debug, insns_size = struct.unpack_from("<HHHHII", data, code_off)
                insns_off = code_off + 16
                units = struct.unpack_from("<%dH" % insns_size, data, insns_off)
                j = 0
                while j < len(units):
                    opcode = units[j] & 0xFF
                    if opcode == 0x1A and j + 1 < len(units):
                        string_idx = units[j + 1]
                        if string_idx < len(strings):
                            refs.append(strings[string_idx])
                        j += 2
                    elif opcode == 0x1B and j + 2 < len(units):
                        string_idx = units[j + 1] | (units[j + 2] << 16)
                        if string_idx < len(strings):
                            refs.append(strings[string_idx])
                        j += 3
                    else:
                        j += 1

            tokens = []
            for s in refs:
                if re.fullmatch(r"[A-Z]{2,8}", s) and s not in tokens:
                    tokens.append(s)
            if 2 <= len(tokens) <= 9:
                return tuple(tokens)
    except (IndexError, struct.error):
        return None
    return None


def extract_k1_magic_from_dex(data: bytes) -> int | None:
    try:
        strings = dex_strings(data)
        if not strings:
            return None
        type_ids_size, type_ids_off = struct.unpack_from("<II", data, 0x40)
        method_ids_size, method_ids_off = struct.unpack_from("<II", data, 0x58)
        class_defs_size, class_defs_off = struct.unpack_from("<II", data, 0x60)
        type_string_ids = [struct.unpack_from("<I", data, type_ids_off + i * 4)[0] for i in range(type_ids_size)]
        methods = [struct.unpack_from("<HHI", data, method_ids_off + i * 8) for i in range(method_ids_size)]
    except struct.error:
        return None

    def skip_fields(off: int, count: int) -> int:
        for _ in range(count):
            _idx, off = read_uleb128(data, off)
            _flags, off = read_uleb128(data, off)
        return off

    def const_values(units: tuple[int, ...]) -> list[int]:
        values = []
        j = 0
        while j < len(units):
            opcode = units[j] & 0xFF
            if opcode == 0x12:
                lit = (units[j] >> 12) & 0xF
                if lit & 8:
                    lit -= 16
                values.append(u32(lit))
                j += 1
            elif opcode == 0x13 and j + 1 < len(units):
                val = units[j + 1]
                if val & 0x8000:
                    val -= 0x10000
                values.append(u32(val))
                j += 2
            elif opcode == 0x14 and j + 2 < len(units):
                val = units[j + 1] | (units[j + 2] << 16)
                values.append(u32(val))
                j += 3
            elif opcode == 0x15 and j + 1 < len(units):
                values.append(u32(units[j + 1] << 16))
                j += 2
            else:
                j += 1
        return values

    try:
        for i in range(class_defs_size):
            class_idx, _access, _super, _interfaces, _source, _annotations, class_data_off, _static_values = struct.unpack_from(
                "<IIIIIIII", data, class_defs_off + i * 32
            )
            desc_idx = type_string_ids[class_idx]
            desc = strings[desc_idx] if desc_idx < len(strings) else ""
            if not desc.endswith("/K1;") or not class_data_off:
                continue

            off = class_data_off
            static_fields_size, off = read_uleb128(data, off)
            instance_fields_size, off = read_uleb128(data, off)
            direct_methods_size, off = read_uleb128(data, off)
            virtual_methods_size, off = read_uleb128(data, off)
            off = skip_fields(off, static_fields_size + instance_fields_size)

            for method_count in (direct_methods_size, virtual_methods_size):
                method_idx = 0
                for _ in range(method_count):
                    diff, off = read_uleb128(data, off)
                    _flags, off = read_uleb128(data, off)
                    code_off, off = read_uleb128(data, off)
                    method_idx += diff
                    if method_idx >= len(methods):
                        continue
                    name_idx = methods[method_idx][2]
                    name = strings[name_idx] if name_idx < len(strings) else ""
                    if name != "g" or not code_off:
                        continue
                    _registers, _ins, _outs, _tries, _debug, insns_size = struct.unpack_from("<HHHHII", data, code_off)
                    units = struct.unpack_from("<%dH" % insns_size, data, code_off + 16)
                    for val in const_values(units):
                        if val != 0x12345678:
                            return val
    except (IndexError, struct.error):
        return None
    return None


def find_zip_inputs(apk_path: str) -> tuple[bytes, bytes, int | None, str | None, str, tuple[str, ...] | None, int | None]:
    with zipfile.ZipFile(apk_path) as zf:
        infos = zf.infolist()
        names = [i.filename for i in infos]

        so_name = choose_member(
            names,
            [
                r"^lib/x86_64/libmobile04\.so$",
                r"^lib/x86_64/.*mobile.*\.so$",
                r"^lib/x86_64/.*\.so$",
            ],
        )
        if not so_name:
            raise ValueError("could not find an x86_64 native library; this solver expects an x86_64 libmobile04.so")

        cache_name = choose_member(
            names,
            [
                r"^assets/appview_cache\.db$",
                r"^assets/.*cache.*\.db$",
                r"^assets/.*\.db$",
            ],
        )
        if not cache_name:
            asset_infos = [i for i in infos if i.filename.startswith("assets/") and 16 <= i.file_size <= 2_000_000]
            if asset_infos:
                cache_name = sorted(asset_infos, key=lambda i: i.file_size)[0].filename
        if not cache_name:
            raise ValueError("could not find the encrypted cache asset")

        mp4_infos = [i for i in infos if i.filename.lower().startswith("res/") and i.filename.lower().endswith(".mp4")]
        raw_len = max((i.file_size for i in mp4_infos), default=None)

        prefix = None
        prefix_rx = re.compile(rb"(flag|ISCC|CTF|DASCTF|NSS|picoCTF|hgame|moectf)[A-Za-z0-9_]*\{", re.I)
        for info in infos:
            if info.file_size > 12_000_000:
                continue
            try:
                data = zf.read(info.filename)
            except Exception:
                continue
            m = prefix_rx.search(data)
            if m:
                prefix = m.group(0).decode("ascii", errors="ignore")
                break

        dex_tokens = None
        dex_magic = None
        for name in names:
            if name.startswith("classes") and name.endswith(".dex"):
                dex_data = zf.read(name)
                dex_tokens = dex_tokens or extract_harp_tokens_from_dex(dex_data)
                dex_magic = dex_magic if dex_magic is not None else extract_k1_magic_from_dex(dex_data)
                if dex_tokens:
                    continue

        return zf.read(so_name), zf.read(cache_name), raw_len, prefix, so_name, dex_tokens, dex_magic


def candidate_sequences(tokens: tuple[str, ...], seq: str | None) -> itertools.chain[tuple[str, ...]]:
    if seq:
        return itertools.chain([(seq,)])
    return ("".join(p) for p in itertools.permutations(tokens))


def solve(apk_path: str, args: argparse.Namespace) -> list[tuple[int, str, str, str, int, int, int]]:
    so_data, cache, guessed_resource_len, guessed_prefix, so_name, dex_tokens, dex_magic = find_zip_inputs(apk_path)
    k1, k2, target, derive_consts, package_from_so, rc6_params = extract_native_constants(so_data)

    package = args.package or package_from_so or DEFAULT_PACKAGE
    magic_mask = dex_magic if dex_magic is not None else DEFAULT_K1_MAGIC
    resource_len = args.resource_len if args.resource_len is not None else guessed_resource_len
    if resource_len is None:
        print("[!] no raw mp4 resource size found; using Java fallback seed 0x12345678", file=sys.stderr)
        derive_seed = 0x12345678
    else:
        derive_seed = u32(java_hash_u32(package) ^ resource_len ^ magic_mask)
    tea_key = tuple(u32(derive_seed ^ c) for c in derive_consts)

    prefix = args.prefix or guessed_prefix or DEFAULT_PREFIX
    target_bytes = normalize_target_bytes(target)
    expected_cipher = repeat_xor(target_bytes, k1)

    if args.tokens:
        tokens = tuple(t.strip() for t in args.tokens.split(",") if t.strip())
    else:
        tokens = dex_tokens or DEFAULT_TOKENS
    if not args.seq and len(tokens) > 9:
        raise ValueError("too many tokens for full permutation; pass --seq or reduce --tokens")

    print(f"[*] native: {so_name}")
    print(f"[*] package: {package}")
    print(f"[*] resource_len: {resource_len if resource_len is not None else 'fallback'}")
    print(f"[*] K1 magic: 0x{magic_mask:08x}")
    print(f"[*] K1={k1!r} K2={k2!r} target={target}")
    print(f"[*] Z1.d constants: {', '.join(hex(x) for x in derive_consts)}")
    print(f"[*] RC6 params: p_xor=0x{rc6_params.p_xor:08x} q_const=0x{rc6_params.q_const:08x} q_mode={rc6_params.q_mode}")
    print(f"[*] prefix: {prefix}")
    token_source = "argv" if args.tokens else ("dex" if dex_tokens else "default")
    print(f"[*] tokens ({token_source}): {', '.join(tokens)}")

    results = []
    total = 0
    for seq_value in candidate_sequences(tokens, args.seq):
        total += 1
        cw = cw_from_sequence(seq_value, tea_key)
        if not cw.startswith("cw:") or "err_len" in cw:
            continue
        decrypted_cache, cache_crc = decrypt_cache(cache, cw)
        seed1 = struct.unpack_from("<I", decrypted_cache, 0x80)[0] if len(decrypted_cache) >= 0x84 else 0
        seed2 = struct.unpack_from("<I", decrypted_cache, 0xB4)[0] if len(decrypted_cache) >= 0xB8 else 0
        schedule = rc6_schedule(seed1, seed2, k2, rc6_params)
        body = rc6_decrypt_block(expected_cipher[:16], schedule) + rc6_decrypt_block(expected_cipher[16:], schedule)
        score = printable_score(body)
        if score < args.min_score:
            continue
        check = rc6_encrypt_block(body[:16], schedule) + rc6_encrypt_block(body[16:], schedule)
        if check != expected_cipher:
            continue
        try:
            body_text = body.decode("ascii")
        except UnicodeDecodeError:
            continue
        results.append((score, prefix + body_text + "}", seq_value, cw, seed1, seed2, cache_crc))

    results.sort(reverse=True, key=lambda item: item[0])
    print(f"[*] tested sequences: {total}")
    return results[: args.max_candidates]


def parse_args(argv: list[str]) -> argparse.Namespace:
    parser = argparse.ArgumentParser(description="Solve mobile04-style Android CTF APK variants.")
    parser.add_argument("apk", help="Path to the APK")
    parser.add_argument("--tokens", help="Comma-separated harp tokens. Auto-detected from DEX when possible")
    parser.add_argument("--seq", help="Test only one already-concatenated harp sequence")
    parser.add_argument("--package", help="Override Java package used by K1.g hash")
    parser.add_argument("--resource-len", type=int, help="Override raw resource byte length used by K1.g")
    parser.add_argument("--prefix", help="Override flag prefix, e.g. ISCC{")
    parser.add_argument("--min-score", type=int, default=100, help="Minimum printable score to keep")
    parser.add_argument("--max-candidates", type=int, default=10, help="Maximum candidates to print")
    return parser.parse_args(argv)


def main(argv: list[str]) -> int:
    args = parse_args(argv)
    try:
        results = solve(args.apk, args)
    except Exception as exc:
        print(f"[!] failed: {exc}", file=sys.stderr)
        return 1

    if not results:
        print("[!] no printable candidates found")
        print("    Try --seq with the exact click sequence, --tokens for changed button codes,")
        print("    or override --package / --resource-len if this APK changed K1.g.")
        return 2

    print()
    for idx, (score, flag, seq, cw, seed1, seed2, cache_crc) in enumerate(results, 1):
        print(f"[{idx}] score={score}")
        print(f"    flag = {flag}")
        print(f"    seq  = {seq}")
        print(f"    cw   = {cw}")
        print(f"    cache_crc=0x{cache_crc:08x} seeds=(0x{seed1:08x}, 0x{seed2:08x})")
    return 0


if __name__ == "__main__":
    raise SystemExit(main(sys.argv[1:]))
